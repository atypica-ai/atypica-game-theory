在学术和用户研究领域，访谈形式有多种分类方法。以下是主要的访谈形式：

## 按结构化程度分类

### 1. **结构化访谈（Structured Interview）**
* 预先设定好所有问题和顺序
* 问题标准化，所有受访者回答相同问题
* 便于量化分析和比较
* 灵活性最低

### 2. **半结构化访谈（Semi-structured Interview）**
* 有访谈提纲，但允许灵活调整
* 可以根据回答追问和深入探讨
* 平衡了标准化和灵活性
* 用户研究中最常用

### 3. **非结构化访谈（Unstructured Interview）**
* 只有大致主题，没有固定问题
* 像自然对话，高度灵活
* 适合探索性研究
* 数据分析难度较大

## 按参与人数分类

### 4. **一对一深度访谈（In-depth Interview / One-on-one Interview）**
* 研究者与单个受访者进行
* 可深入探讨个人经验和观点
* 保密性好，适合敏感话题

### 5. **焦点小组访谈/座谈会（Focus Group）**
* 6-12人的小组讨论
* 观察群体互动和共识形成
* 能快速收集多元观点
* 注意群体思维和主导者影响

### 6. **配对访谈（Dyadic Interview）**
* 同时访谈两个人
* 适合研究关系、互动或共同经验
* 如夫妻、搭档等

## 按实施方式分类

### 7. **面对面访谈（Face-to-face Interview）**
* 传统方式，信息最丰富
* 可观察肢体语言和表情

### 8. **电话访谈（Telephone Interview）**
* 成本低，覆盖范围广
* 缺少视觉信息

### 9. **在线访谈（Online Interview）**
* 视频访谈（如Zoom）
* 即时通讯文字访谈
* 异步邮件访谈

## 特殊类型访谈

### 10. **关键事件访谈（Critical Incident Technique）**
* 聚焦于特定的重要事件
* 让受访者回忆具体情境和行为
* 适合发现问题和需求

### 11. **情境访谈（Contextual Interview）**
* 在用户实际使用环境中进行
* 边观察边访谈
* 结合了观察法和访谈法

### 12. **回溯式访谈（Retrospective Interview）**
* 让用户回顾过去的经历
* 常配合日志或记录使用

### 13. **即时出声思维访谈（Think-aloud Interview）**
* 用户边使用产品边说出想法
* 实时捕捉认知过程
* 常用于可用性测试

### 14. **专家访谈（Expert Interview）**
* 访谈领域专家
* 获取专业知识和见解
* 常用于研究初期

### 15. **民族志访谈（Ethnographic Interview）**
* 长期深入用户生活环境
* 理解文化和社会背景
* 时间成本高

### 16. **叙事访谈（Narrative Interview）**
* 鼓励受访者讲故事
* 了解个人经历的意义建构
* 开放式，最小干预

### 17. **阶梯访谈（Laddering Interview）**
* 通过"为什么"不断追问
* 从具体属性到抽象价值
* 揭示深层动机

## 混合方法

### 18. **日记研究后访谈（Diary Study Follow-up Interview）**
* 先让用户记录日志
* 再进行访谈深入探讨

### 19. **问卷后跟进访谈（Survey Follow-up Interview）**
* 基于问卷结果进行深入访谈
* 解释量化数据背后的原因

---

## 选择建议

选择访谈形式时考虑：
* **研究目标**：探索性 vs 验证性
* **样本规模**：深度 vs 广度
* **资源限制**：时间、预算、人力
* **话题敏感度**：隐私性要求
* **受访者特征**：可及性、舒适度

在实际用户研究中，常常会组合使用多种方法，以获得更全面的洞察。
