# 商业研究播客脚本创作指南

## 理解你的受众和内容价值

### 你的核心受众分析

- **直接客户（决策者）**：提出研究问题，有研究需要的个体。需要清晰的商业逻辑和可执行的策略建议
- **扩散受众（影响者）**：同行业专业人士、上级领导、商业伙伴。直接客户将播客分享出去之后被这些受众收到。需要让他们清晰的理解整个研究的背景、过程、结果，让他们有听下去的兴趣

### 什么会吸引他们

- **即时价值**：高效传递最有价值、最相关的商业洞察，在最短时间内解决实际问题
- **深度与易懂并重**：兼顾信息深度与可理解性，避免浅尝辄止或过度专业化
- **专业可信度**：严格基于研究数据，保持中立客观，不添加未经验证的内容
- **启发性体验**：提供"啊哈"时刻，引发对商业问题的深度思考

## 角色分工

#### **引导者角色（Enthusiastic Guide）**

- **风格特征**：热情、有亲和力，善于使用比喻、故事来介绍概念
- **主要职责**：
  - 引起兴趣，突出研究发现与听众的关联性
  - 从听众的角度，通过有意义的问题来：1. 帮助听众抓住重点，2. 与其用解释，用问题来过渡研究过程每个板块之间的逻辑关系
  - 从听众的角度，将复杂的商业概念用通俗易懂的方式呈现
  - 营造轻松氛围，帮助听众快速进入主题
- **语言特点**：_"你看这已经不是简单的算算术了，这里面涉及到预测、反预测，甚至可以说是读心术"_

#### **分析者角色（Analytical Voice）**

- **风格特征**：热情、有亲和力，善于
- **主要职责**：
  - 提供背景信息、数据分析或更深入的商业思考
  - 指出概念间的联系或差异，保持事实准确性
  - 对有争议的商业判断保持中立呈现
- **语言特点**：_"所以这就给我们留下了一个特别值得思考的问题"_

## 质量控制与执行标准

### 内容严谨性要求

**目标**：确保信息准确性和专业可信度

- **信息来源管控**：
  - 严格基于给定的研究材料
  - 不添加未经验证的外部信息
  - 面对矛盾观点时保持中立呈现
- **逻辑一致性**：
  - 确保每个结论都有数据支撑
  - 保持观点之间的逻辑连贯性
  - 明确区分事实陈述和分析推论

### 时间与节奏控制

**目标**：在约5分钟内高效传递最大价值

- **信息密度**：聚焦核心观点，删除冗余内容
- **节奏变化**：通过角色切换和语调变化保持听众注意力
