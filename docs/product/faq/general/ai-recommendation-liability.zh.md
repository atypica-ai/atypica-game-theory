## 问题类型
产品 Q&A

---

1. 如果我根据AI的建议做了决策，结果失败了，atypica会赔偿吗？

## 核心回答

**明确责任边界：atypica是研究工具，不是决策替代。**

**核心原则**：
1. **atypica的定位**：辅助决策工具，提供洞察和参考，不替代你的判断
2. **用户责任**：最终决策权和责任都在你，AI是你的研究助手
3. **风险控制**：我们提供多种机制帮你验证AI建议的可靠性
4. **服务条款**：明确界定使用范围和免责条款（见用户协议）

**类比**：
```
就像Excel是财务分析工具，但不对你的财务决策负责。
atypica是市场研究工具，但不对你的商业决策负责。
```

**如何降低风险**：
- ✅ 多轮验证：用Scout + Interview + Discussion交叉验证
- ✅ 人类审核：关键决策要结合人类洞察
- ✅ 小规模测试：先小范围测试，再全面推广
- ✅ 追溯来源：AI建议都有数据来源和推理过程

**价值**：

### 3. 如何正确使用AI建议

#### 正确的使用流程

**Step 1: 获取AI建议**
```
用Scout/Interview/Discussion研究市场
  ↓
AI生成报告和建议
```

**Step 2: 理解推理过程**
```
阅读AI的推理逻辑：
- 数据来源是什么？
- 推理过程是否合理？
- 有哪些不确定因素？
```

**Step 3: 交叉验证**
```
方法1：用不同的AI工具验证（Scout + Interview）
方法2：用真人小范围验证（3-5人）
方法3：查看竞品和行业数据
```

**Step 4: 结合自身情况判断**
```
考虑你的特殊因素：
- 成本结构：你的成本支持这个定价吗？
- 品牌定位：符合你的品牌调性吗？
- 市场环境：当前市场环境适合吗？
- 资源能力：你有能力执行吗？
```

**Step 5: 小规模测试**
```
先小范围试点：
- 不要立即全面推广
- 先测试100人，看真实反馈
- 根据反馈调整策略
```

**Step 6: 全面推广**
```
测试成功后再全面推广：
- 降低风险
- 提升成功率
```
