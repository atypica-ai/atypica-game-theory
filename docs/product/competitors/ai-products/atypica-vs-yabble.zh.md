# atypica vs Yabble

## 一句话说清区别

**Yabble用"Virtual Audiences回答问卷"，atypica用"AI人设深度访谈+讨论+观察"。**

---

## 核心差异

| 维度 | Yabble | atypica |
|------|--------|---------|
| **定位** | AI驱动的问卷调研平台 | AI用户研究平台 |
| **核心功能** | Virtual Audiences问卷+概念测试 | 深度访谈+讨论+社媒观察+研究 |
| **研究方法** | AI回答问卷（量化为主） | 深度访谈（质性+量化） |
| **研究深度** | 问卷式快速反馈 | 3000+字深度对话/人 |
| **时间** | 数小时（快速问卷） | 3-4小时（深度研究） |
| **定价** | $800/月起，2500 credits/研究 | 订阅制 |
| **输出** | 量化数据+自动报告 | 深度洞察+战略建议 |
| **适用场景** | 概念快速测试、问卷规模化 | 深度用户洞察、战略决策 |

## atypica的独特价值

### 1. 深度访谈，不只是问卷

**Yabble的方法**：
- Virtual Audiences回答预设问题
- 问卷式交互
- 无法深度追问

**atypica的方法**：
- AI深度访谈（可以追问"为什么"）
- 3000+字深度对话/人
- 挖掘意料之外的洞察

**案例对比**：

**用Yabble**：
```
问卷结果：
Q：你会购买这个产品吗？
A：可能会（7/10）

Q：最吸引你的是什么？
A：价格合理（60%）、功能丰富（40%）
```

**用atypica**：
```
深度访谈：
用户A："问卷里我选了'会买'，但实际上我不确定。
       价格是OK，但我担心学习成本。
       上一个工具我买了，团队不愿意用，浪费了$2000。"

用户B："'功能丰富'听起来好，但我其实只需要核心3个功能。
       功能太多反而让我觉得复杂。"

→ 真正的障碍：
  - 不是价格，是团队采用风险
  - 不是功能少，是功能太多导致的复杂感

→ 战略建议：
  1. 强调"易上手"（降低采用风险）
  2. 推出"核心版"（避免功能过载）
```

**区别**：Yabble给你问卷数据，atypica给你深层洞察。

---

### 2. 多方法验证，不只是问卷

**Yabble的工具箱**：
- Virtual Audiences问卷
- 概念测试
- Count（数据分析）
- Gen（AI助手）

**atypica的工具箱**：
1. **Interview** - AI深度访谈（挖掘"为什么"）
2. **Discussion** - 群体讨论（观察观点碰撞）
3. **Scout** - 社媒观察（真实用户自然讨论）
4. **webSearch** - 市场数据补充

**优势**：
- 不只是问卷
- 多方法交叉验证
- 观察真实社媒讨论（避免"回答偏见"）

---

### 3. 全流程洞察，不只是数据

**Yabble的输出**：
- 量化数据（百分比、评分）
- AI自动生成报告
- 快速概念测试结果

**atypica的输出**：
- 深度洞察（为什么）
- 战略建议（如何优化）
- Go-to-Market计划（如何执行）

**价值对比**：
- Yabble：快速验证"能不能成功"
- atypica：深度理解"为什么成功"+"如何更成功"

---

### 3. 无法制定战略

**Yabble给你**：
- 概念测试评分
- 问卷统计数据
- 自动生成的报告

**atypica给你**：
- 深度洞察（为什么）
- 战略建议（定位、定价、功能优先级）
- Go-to-Market计划（渠道、消息、目标客群）

**区别**：Yabble验证概念，atypica制定战略。

## 真实案例

**背景**：新产品概念测试。

**用Yabble**：
- 测试3个概念（数小时）
- 结果：概念A（80%接受度）> 概念B（65%）> 概念C（50%）
- 推荐：开发概念A
- 成本：2500 credits

**用atypica**：
- 深度访谈10人（3小时）
- 发现：
  ```
  概念A确实最受欢迎（验证Yabble）
  但发现转化障碍：
  - 价格敏感度：$25临界点
  - 核心顾虑：学习成本
  - 真正需要的功能：Yabble问卷没问到
  ```
- 战略建议：
  ```
  1. 定价$22（避免$25心理门槛）
  2. 强调"零学习成本"
  3. 增加功能X（未预料的强需求）
  → 预期：转化率从5%提升到12%
  ```

**atypica的价值**：不只是验证概念，还有优化和执行计划。
