# atypica vs Qualtrics

## 一句话说清区别

**Qualtrics是"企业级全能调研平台"（问卷+panel+复杂分析），atypica是"AI用户研究平台"（深度洞察+快速决策）。**

---

## 核心差异

| 维度 | Qualtrics | atypica |
|------|-----------|---------|
| **定位** | 企业级体验管理平台 | AI用户研究平台 |
| **核心能力** | 复杂问卷+panel招募+高级分析 | AI深度访谈+讨论+观察 |
| **研究方法** | 问卷为主（量化） | 深度访谈为主（质性+量化） |

| **目标用户** | 大企业、专业研究团队 | 全规模团队，快速决策 |
| **用户类型** | 真人panel | AI人设 |

## atypica的独特价值

### 1. 深度洞察，不只是问卷数据

**Qualtrics的局限**：
- 以问卷为主（量化数据）
- 无法深度追问"为什么"
- 需要提前设计好所有问题

**atypica的能力**：
- AI深度访谈（可以追问"为什么"）
- 3000+字深度对话/人
- 挖掘意料之外的洞察

**案例对比**：

**用Qualtrics**：
```
问卷结果：
Q：你为什么流失？
A1：价格太贵（40%）
A2：功能不够（30%）
A3：体验不好（20%）
A4：其他（10%）

→ 知道表层原因，但不知道"为什么"
```

**用atypica**：
```
深度访谈：
用户A："不是价格贵，是性价比低。
       $50/月，但我只用到20%功能，
       其他80%功能我根本用不上。"

→ 真正的问题：功能臃肿，不是定价高
→ 解决方案：推出$20/月轻量版（核心功能）
```

**区别**：Qualtrics告诉你"what"，atypica告诉你"why"和"how to fix"。

---

### 2. 快速决策，无需复杂设置

**Qualtrics的复杂度**：
- 学习曲线陡峭（需要培训）
- 设计复杂问卷（需要专业技能）
- 高级分析功能（需要统计知识）
- 企业级设置（权限、流程、合规）

**atypica的简单性**：
- Plan Mode自动设计研究
- 无需设计问卷
- 无需统计知识
- 3-4小时完成完整研究

**适用场景**：
- 快速决策环境
- 没有专职研究团队
- 预算和时间有限
- 需要频繁做研究

## 真实案例

**背景**：产品团队想理解用户流失原因。

**用Qualtrics**：
- Week 1：设计流失问卷
- Week 2-3：招募用户，收集数据
- Week 4：数据分析
- 结果："价格太贵"（40%）、"功能不够"（30%）
- 成本：$420 + panel费 + 4周时间

**用atypica**：
- Day 1：深度访谈10个流失用户（3小时）
- 发现：
  ```
  不是"价格贵"，而是"性价比低"
  用户只用20%功能，其他80%用不上
  真正需要的功能反而缺失
  ```
- 战略建议：
  ```
  1. 推出$20轻量版（核心功能）
  2. 增加用户真正需要的功能X和Y
  3. 预期：留存率提升40%
  ```
- 成本：订阅费 + 3小时

**atypica的价值**：不只是知道表层原因，找到根本问题和解决方案。
