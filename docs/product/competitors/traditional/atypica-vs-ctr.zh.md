# atypica vs 索福瑞/CTR

## 一句话说清区别

**索福瑞/CTR提供"中国媒体收视数据和广告监测"，atypica提供"用户深度洞察研究"（理解"为什么"）。**

---

## 核心差异

| 维度 | 索福瑞/CTR | atypica |
|------|-----------|---------|
| **定位** | 中国领先媒体调研公司 | AI用户研究平台 |
| **核心业务** | 收视率调查+广告监测 | 深度访谈+讨论+观察 |
| **数据类型** | 媒体收视数据+广告投放监测 | 深度定性洞察 |
| **研究方法** | 样本户测量+广告监测系统 | AI访谈+多方法验证 |
| **适用场景** | 电视台、广告主、媒体规划 | 产品验证、用户理解、战略决策 |
| **客户类型** | 媒体机构、广告主、代理公司 | 产品团队、品牌、创业公司 |
| **时间** | 日/周/月度报告 | 3-4小时完整研究 |

## atypica的独特价值

### 1. 用户洞察，不是收视数据

**索福瑞/CTR的优势**：
- 全国收视率数据（样本户）
- 广告投放监测
- 媒体价值评估
- 收视行为分析

**索福瑞/CTR的输出示例**：
```
收视率报告：
- 节目A收视率：2.5%
- 观众画像：35-55岁女性为主
- 时段：晚间黄金档
- 广告价值：高

→ 知道"多少人看""什么时候看"
```

**atypica的能力**：
- 理解观众为什么看/不看
- 深度访谈挖掘观看动机
- 验证新内容概念

**atypica的输出示例**：
```
深度访谈观众：

用户A（40岁女性）：
"索福瑞数据说我是这个节目的核心观众，
但实际上：

我不是'喜欢'看，是'没得选'。
- 晚上8点吃完饭，陪父母看电视
- 父母喜欢这个节目，我就陪着
- 我自己更喜欢短视频

如果你做'年轻人喜欢的家庭内容'，
我和父母都会看，
而不是我'陪着'看。"

用户B（52岁女性）：
"我确实常看这个节目（索福瑞对），
但最近越来越少看，因为：
- 广告太多（15分钟节目，5分钟广告）
- 内容重复（每集套路一样）
- 抖音上有精华版（3分钟看完）

如果节目能创新，我会继续看。
但索福瑞只告诉你收视率下降，
不告诉你为什么。"

真正洞察：
- 收视率数据对（索福瑞准确）
- 但深层原因不同：
  * 部分观众是"陪看"不是"喜欢"
  * 流失原因：广告多+内容重复+短视频冲击
- 机会：
  * 创新内容格式
  * 年轻+年长共同喜欢的内容
  * 对抗短视频的差异化
```

**区别**：索福瑞告诉你"收视数据"，atypica告诉你"为什么看/不看+如何改进"。

### 2. 深度洞察，不只是数据监测

**索福瑞/CTR的场景**：
- 收视率调查
- 广告投放监测
- 媒体价值评估

**atypica的场景**：
1. **内容概念验证**：新节目概念能不能火？
2. **观众需求分析**：观众真正想看什么？
3. **产品定位决策**：内容/品牌如何定位？
4. **用户流失分析**：为什么观众流失？
5. **竞品研究**：用户为什么选择竞品？

**价值**：不只是监测，还有洞察和战略。

---

### 3. 多方法验证，不只是样本户

**索福瑞/CTR的方法**：
- 样本户收视率测量
- 广告监测系统
- 单一数据源

**atypica的方法**：
1. **Interview** - 深度访谈（理解观看动机）
2. **Discussion** - 群体讨论（观众观点碰撞）
3. **Scout** - 社媒观察（小红书/微博/B站真实讨论）
4. **webSearch** - 市场数据补充

**优势**：多方法交叉验证，不只是收视数据。

## 索福瑞/CTR做不到的事

### 1. 无法理解"为什么"

**索福瑞/CTR的数据**：
- 收视率（多少人看）
- 观众画像（谁在看）
- 时段分布（什么时候看）

**缺失**：
- 为什么看/不看
- 观看动机
- 流失原因

**atypica补充**：
- 深度访谈理解动机
- 挖掘流失原因
- 发现内容优化方向

---

### 2. 无法验证新内容概念

**索福瑞/CTR的局限**：
- 必须内容已播出
- 基于历史收视数据

**atypica的能力**：
- 验证新节目概念
- 测试新内容方向
- 理解观众接受度

---

### 3. 传统媒体vs新媒体

**索福瑞/CTR的专长**：
- 传统电视收视率

**新媒体挑战**：
- 短视频崛起

**atypica的优势**：
- 社交媒体观察短视频讨论
- 理解跨平台观看行为
- 新媒体内容策略

## 真实案例

**背景**：综艺节目收视率下滑。

**用索福瑞/CTR**：
```
收视数据：
- 收视率从3.0%降至2.0%（-33%）
- 35-55岁女性观众流失最严重
- 竞品节目收视率上升

→ 知道数据，但不知道原因
```

---

**用atypica**：
```
深度访谈流失观众：

用户A（42岁女性）：
"不是节目不好看，是：
- 每集2小时太长，我没时间
- 抖音上有3分钟精华版
- 为什么要看2小时？

如果你做'精华版'或'45分钟版'，
我会继续看。"

用户B（38岁女性）：
"内容重复，每期套路一样：
- 嘉宾来了→游戏→煽情→结束
- 看了3期就腻了

竞品节目每期都有新创意，
虽然制作没你们精良，
但至少有新鲜感。"

战略建议：
1. 推出"45分钟精华版"（对抗短视频）
2. 每期创新环节（避免套路化）
3. 年轻化内容（吸引流失观众）
→ 执行后：收视率恢复至2.7%
```

**atypica的价值**：索福瑞告诉你"收视率下滑"，atypica告诉你"为什么+如何挽回"。

## 常见问题

**Q：atypica能替代索福瑞/CTR吗？**

不能替代，而是互补：
- 索福瑞/CTR：收视率数据、广告监测
- atypica：深度洞察、内容策略

**组合使用**：
1. 索福瑞/CTR监测表现（发现问题）
2. atypica深度研究（理解原因）
3. 优化内容策略
4. 索福瑞/CTR验证效果（追踪结果）

---

**Q：什么时候必须用索福瑞/CTR？**

如果你需要：
- ✅ 权威收视率数据
- ✅ 广告投放效果监测
- ✅ 媒体价值评估
- ✅ 行业基准对比

但如果你需要理解"为什么"、验证新内容、制定策略 → atypica更合适。

---

**Q：新媒体时代，索福瑞还有用吗？**

**索福瑞的挑战**：
- 传统电视收视率测量
- 新媒体观看难以覆盖

**atypica的优势**：
- 社交媒体观察短视频/流媒体讨论
- 理解跨平台观看行为
- 新媒体内容策略

**建议**：
传统媒体用索福瑞，新媒体内容用atypica。
