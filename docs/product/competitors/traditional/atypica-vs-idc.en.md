## atypica's Unique Value

### 1. User Insights, Not Market Data

**IDC's strengths**:
- Global IT market size data
- Technology adoption rates and trends
- MarketScape vendor evaluation
- IT spending forecasts

**IDC's output example**:
```
IDC Report:
"2025 Global Cloud Computing Market"

Market size: $480B (2025)
Growth rate: +19% YoY
Major players: AWS (32%), Azure (22%), GCP (10%)
Technology trends: Multi-cloud, edge computing, AI integration

→ Know "market size" and "trends"
```

**atypica's capability**:
- Understand why users choose/don't choose cloud services
- Deep interviews dig into decision barriers
- Validate new product concepts

**atypica's output example**:
```
Deep interviews with cloud service users:

User A (CTO, 50-person company):
"IDC says cloud market growing 19%,
but we're still using on-premise servers.

Not that I don't know cloud benefits,
real barriers are:
- Migration cost: Need to refactor applications
- Data compliance: We're in financial industry
- Cost opacity: Can't understand AWS bills
- Vendor lock-in: Fear of being trapped

If there's cloud service that solves these concerns,
I'd consider. But IDC reports won't tell you
why 80% of SMBs haven't moved to cloud yet."

User B (Technical Manager, 15-person team):
"Multi-cloud sounds good, but actually:
- We don't have dedicated DevOps
- Managing 2 cloud platforms is too complex
- We need 'one-click migration' simple solution

You're doing complex multi-cloud solutions, not for us.
Do 'simplified cloud migration,' we'll buy."

Real insights:
- Market is indeed growing (IDC correct)
- But different segment barriers vary:
  * Financial: Compliance concerns
  * SMBs: Migration cost + complexity
  * No DevOps teams: Management difficulty
- Opportunity: "Niche cloud services" targeting specific barriers
```

**The difference**: IDC tells you "market is growing," atypica tells you "why many haven't adopted + how to win them."

## Real-World Case

**Background**: Cloud service startup, positioning decision.

**Using IDC**:
```
IDC Report:
"Cloud market growing 19%, multi-cloud trend obvious"

→ Decide to build "multi-cloud management platform"
→ Develop for 6 months
→ After launch: Customer acquisition difficult, users say "too complex"
```

---

**Using atypica (before development)**:
```
Deep interviews with target users:

Findings:
- Market is indeed growing (IDC correct)
- But "multi-cloud" demand mainly in large enterprises
- SMBs (80% of market) actually need:
  * "Simplified cloud migration" (one-click migration)
  * "Cost transparency" (predict bills)
  * "Compliance assurance" (finance/healthcare)

Strategy adjustment:
- Don't do "multi-cloud management" (crowded large enterprise market)
- Do "SMB dedicated cloud" (niche opportunity)
- Features: "5-minute cloud migration" + "cost prediction"
→ 3 months after launch, 300+ paying users
```

**atypica's value**: IDC tells you "market trends," atypica helps you find "niche opportunities."
