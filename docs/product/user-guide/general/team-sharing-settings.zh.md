# 如何和团队共享人设库和研究项目？

## 问题类型
用户手册问题

---

## 快速回答

**团队版自动共享，个人版不支持共享。**

**团队版（¥1,999/月）**：
- ✅ **人设库自动共享**：创建人设时选择「团队人设库」
- ✅ **研究项目可选共享**：创建时选择「团队可见」或后续手动共享
- ✅ **Sage专家自动共享**：创建的Sage专家全团队可用
- ✅ **权限控制**：可以设置只读/可编辑权限

**个人版（免费/专业版）**：
- ❌ **不支持共享**：只能自己使用，无法共享给同事
- 如需共享 → 升级到团队版

## 详细说明

### 共享人设库

#### 创建团队人设

**操作流程**：

```
Step 1: 创建新人设
→ 进入「人设库」
→ 点击「创建人设」按钮

Step 2: 选择共享范围
→ 在创建表单中，选择「存储位置」：
  □ 我的私有人设库（只有你能用）
  ✅ 团队人设库（全团队可用）

Step 3: 创建人设
→ 选择创建方式：
  - 从公共库选择
  - 上传访谈记录（自定义人设）
  - 使用Scout生成（自定义人设）
→ 完成创建

Step 4: 团队成员可见
→ 所有团队成员在人设选择器中
→ 可以看到这个人设
→ 可以在Interview/Discussion中使用
```

---

#### 使用团队人设

**团队成员使用**：

```
Step 1: 创建Interview/Discussion
→ 选择人设

Step 2: 切换到「团队人设库」标签
→ 看到所有团队共享的人设
→ 可以按标签、行业、年龄等筛选

Step 3: 选择人设
→ 点击人设卡片
→ 查看详细信息
→ 加入访谈

价值：
✅ 不需要重复创建人设
✅ 使用高质量的自定义人设人设
✅ 团队标准统一
```

---

#### 团队人设管理

**查看团队人设**：

```
位置：人设库 → 团队人设

显示内容：
- 人设头像和名字
- 创建者
- 创建时间
- 使用次数
- 标签和分类

示例：
Linda（28岁，产品经理）
创建者：张伟  2026-01-15
使用次数：15次
标签：价格敏感、科技爱好者
```

---

**编辑团队人设**：

```
权限：
✅ 创建者：可以编辑和删除
✅ 管理员：可以编辑和删除
⚠️ 普通成员：可以查看和使用，不能编辑

操作：
Step 1: 进入团队人设库
Step 2: 选择要编辑的人设
Step 3: 点击「编辑」按钮
Step 4: 修改信息
Step 5: 保存
```

---

**删除团队人设**：

```
警告：删除团队人设会影响所有使用该人设的研究

操作：
Step 1: 进入团队人设库
Step 2: 选择要删除的人设
Step 3: 点击「删除」按钮
Step 4: 确认删除

确认对话框：
「确定要删除 [人设名] 吗？
 - 该人设已被15个研究使用
 - 删除后，这些研究中的人设仍会保留历史记录
 - 但无法再创建新研究使用该人设
 - 此操作不可撤销」
```

### 共享研究项目

#### 创建团队可见研究

**默认行为**：
```
团队版：
- 新创建的研究默认为「团队可见」
- 所有团队成员都能看到
- 可以在创建时选择「私有」

个人版：
- 所有研究都是私有的
- 不支持共享
```

---

**创建时共享**：

```
Step 1: 创建新研究
→ 进入Plan Mode
→ 描述研究目标

Step 2: 选择可见范围
→ 在研究设置中：
  ✅ 团队可见：所有成员可以查看
  □ 私有：只有你能看到

Step 3: 设置权限（可选）
→ 仅查看：只能查看研究内容
→ 可编辑：可以继续追问和生成报告
→ 默认：可编辑

Step 4: 开始研究
→ 团队成员立即可以看到这个研究
→ 可以实时查看进度
```

---

**后续共享已有研究**：

```
如果研究创建时是私有的，可以后续共享：

Step 1: 打开私有研究
Step 2: 点击右上角「共享」按钮
Step 3: 选择共享范围和权限：
- 共享给：全团队 / 特定成员
- 权限：仅查看 / 可编辑
Step 4: 确认共享

共享后：
✅ 选定的成员可以访问该研究
✅ 在他们的「我的研究」中显示
✅ 标记为「共享」
```

---

**团队研究列表**：

```
位置：我的研究 → 团队研究

显示内容：
- 所有团队可见的研究
- 创建者
- 创建时间
- 研究状态（进行中/已完成）
- 你的权限（查看/编辑）

筛选和排序：
✅ 按创建者筛选
✅ 按时间排序
✅ 按状态筛选
✅ 搜索关键词

示例：
健康零食用户需求
创建者：张伟  2026-01-20  已完成  可编辑

包装设计测试
创建者：李娜  2026-01-25  进行中  仅查看
```

#### 协同编辑研究

**多人协同**：

```
功能：
✅ 多人可以同时查看同一个研究
✅ 具有编辑权限的成员可以追问AI
✅ 实时同步更新
⚠️ 不支持同时编辑（避免冲突）

使用场景：
- 产品经理创建研究
- 市场同事查看和讨论
- 设计师基于研究结果设计
```

```
功能：
✅ 在研究中添加评论
✅ @提及团队成员
✅ 讨论研究发现
✅ 记录决策和行动

操作：
Step 1: 打开共享研究
Step 2: 在任意位置点击「评论」按钮
Step 3: 输入评论内容
- 可以@提及成员：@张伟
```

### 共享Sage专家

#### 创建团队Sage专家

**操作流程**：

```
Step 1: 创建新Sage专家
→ 进入「Sage」功能
→ 点击「创建专家」

Step 2: 选择共享范围
→ 存储位置：
  □ 我的私有专家
  ✅ 团队专家

Step 3: 上传知识源
→ 上传公司文档（设计规范、产品文档等）
→ AI分析并构建Memory Document

Step 4: 完成创建
→ 团队专家立即对全团队可用
→ 所有成员都能咨询这个专家
```

---

#### 使用团队Sage专家

```
Step 1: 进入Sage功能

Step 3: 选择专家并咨询
→ 点击专家卡片
→ 开始对话
→ 咨询专业建议

示例团队Sage专家：
1. UX设计专家：基于公司设计系统v2.3
2. 产品策略专家：基于历史产品决策
3. 技术架构专家：基于技术文档
4. 营销专家：基于营销案例库
```

```
团队Sage的价值：持续进化
```

### 权限管理

#### 3种权限级别

**仅查看**：
```
可以：
✅ 查看研究内容
✅ 查看对话记录
✅ 查看报告
✅ 导出报告

不可以：
❌ 不能追问AI
❌ 不能编辑研究
❌ 不能生成新报告
❌ 不能删除研究

适用场景：
- 高管查看研究成果
- 外部顾问临时查看
- 存档的历史研究
```

---

**可编辑**：
```
可以：
✅ 所有「仅查看」权限
✅ 追问AI
✅ 生成新报告
✅ 添加评论

不可以：
❌ 不能删除研究（只有创建者和管理员）
❌ 不能修改研究设置

适用场景：
- 团队协作研究
- 产品经理之间共享
- 持续深挖洞察
```

---

**管理员**：
```
可以：
✅ 所有「可编辑」权限
✅ 删除研究
✅ 修改研究设置
✅ 转移所有权

适用场景：
- 团队管理员
- 部门负责人
```

---

#### 修改权限

**操作流程**：

```
Step 1: 打开共享研究
Step 2: 点击「共享设置」按钮
Step 3: 查看当前权限列表

示例：
张伟（创建者）：管理员
李娜：可编辑
王芳：仅查看

Step 4: 修改特定成员权限
→ 点击成员旁的「编辑」
→ 选择新权限
→ 保存

Step 5: 添加新成员
→ 点击「添加成员」
→ 选择团队成员
→ 设置权限
→ 确认
```

## 常见问题

### Q1：个人版可以共享吗？

**不可以，只有团队版支持共享。**

**版本对比**：
```
免费版/专业版：
❌ 不支持共享人设库
❌ 不支持共享研究项目
❌ 不支持共享Sage专家

团队版：
✅ 支持共享人设库
✅ 支持共享研究项目
✅ 支持共享Sage专家

如需共享 → 升级到团队版（¥1,999/月）
```

---

### Q2：共享后可以撤销吗？

**可以，随时撤销。**

**操作**：
```
Step 1: 打开共享的研究/人设
Step 2: 点击「共享设置」
Step 3: 移除特定成员 或 改为私有
Step 4: 确认

撤销后：
✅ 被移除的成员立即失去访问权限
✅ 研究/人设对他们不可见
```

---

### Q3：共享的研究可以设置密码吗？

**不支持密码保护，但有权限控制。**

**权限控制**：
```
更安全的方式：
✅ 只共享给特定成员（不是全团队）
✅ 设置「仅查看」权限（不能编辑）
✅ 定期审查共享设置
✅ 敏感研究设置为私有

不支持：
❌ 密码保护
❌ 外部链接分享（只能团队内）
```

---

### Q4：共享人设会消耗我的额度吗？

**不会，团队共享额度。**

**团队版额度**：
```
团队版：
✅ 无限制Interview/Scout/报告
✅ 全团队共享额度（不是按人分配）
✅ 不管谁使用，都从团队额度扣除

案例：
- 团队有5个成员
- 李娜用团队人设做10次Interview
- 张伟用团队人设做15次Interview
- 都从团队额度扣除，不单独计费
```

---

### Q5：删除团队人设会影响历史研究吗？

**不会影响历史研究，但无法创建新研究使用。**

**删除人设后**：
```
历史研究：
✅ 已使用该人设的研究保留完整记录
✅ 对话历史不受影响
✅ 可以继续查看

新研究：
❌ 无法选择该人设
❌ 无法创建新访谈使用该人设

建议：
- 不要随意删除被广泛使用的人设
- 删除前检查使用次数
- 考虑「归档」而非删除
```

---

### Q6：可以和外部合作伙伴共享吗？

**不支持外部共享，只能团队内。**

**当前限制**：
```
只能共享给：
✅ 团队成员（已加入团队的同事）

不能共享给：
❌ 外部合作伙伴
❌ 客户
❌ 顾问（除非邀请加入团队）
```

**替代方案**：
```
如果需要和外部分享：
1. 导出报告（Markdown/PDF/Word）
2. 通过邮件或其他方式分享
3. 外部人员查看导出的报告

未来可能支持：
- 外部链接分享（roadmap）
- 密码保护分享
```

---

### Q7：共享设置会保留多久？

**永久保留，直到你修改或删除。**

**保留机制**：
```
共享设置：
✅ 永久有效
✅ 不会自动过期
✅ 不会因为成员离职自动调整

如果需要修改：
- 手动进入共享设置
- 调整权限或移除成员
```
