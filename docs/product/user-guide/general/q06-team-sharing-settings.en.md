# How to Share Persona Library and Research Projects with Team?

## Question Type
User Manual Question

---

## Quick Answer

**Team version automatically shares, personal version does not support sharing.**

**Team Version (¥1,999/month)**:
- ✅ **Persona Library Auto-Shared**: Select "Team Persona Library" when creating personas
- ✅ **Research Projects Optional Sharing**: Select "Team Visible" when creating or manually share later
- ✅ **Sage Experts Auto-Shared**: Created Sage experts available to entire team
- ✅ **Permission Control**: Can set read-only/editable permissions

**Personal Version (Free/Professional)**:
- ❌ **No Sharing Support**: Only for personal use, cannot share with colleagues
- To share → Upgrade to Team version

## Detailed Explanation

### Sharing Persona Library

#### Creating Team Personas

**Operation Flow**:

```
Step 1: Create New Persona
→ Go to "Persona Library"
→ Click "Create Persona" button

Step 2: Select Sharing Scope
→ In creation form, select "Storage Location":
  □ My Private Persona Library (only you can use)
  ✅ Team Persona Library (entire team can use)

Step 3: Create Persona
→ Choose creation method:
  - Select from public library
  - Upload interview transcripts (custom persona)
  - Use Scout to generate (custom persona)
→ Complete creation

Step 4: Visible to Team Members
→ All team members can see this persona
→ In the persona selector
→ Can use in Interview/Discussion
```

---

#### Using Team Personas

**Team Member Usage**:

```
Step 1: Create Interview/Discussion
→ Select persona

Step 2: Switch to "Team Persona Library" Tab
→ See all team-shared personas
→ Can filter by tags, industry, age, etc.

Step 3: Select Persona
→ Click persona card
→ View detailed information
→ Add to interview

Value:
✅ No need to repeatedly create personas
✅ Use high-quality custom personas
✅ Unified team standards
```

```

### Sharing Research Projects

#### Creating Team-Visible Research

**Default Behavior**:
```
Team Version:
- New research defaults to "Team Visible"
- All team members can see
- Can select "Private" during creation

Personal Version:
- All research is private
- No sharing support
```

---

**Share During Creation**:

```
Step 1: Create New Research
→ Enter Plan Mode
→ Describe research objectives

Step 2: Select Visibility Scope
→ In research settings:
  ✅ Team Visible: All members can view
  □ Private: Only you can see

Step 3: Set Permissions (Optional)
→ View Only: Can only view research content
→ Editable: Can continue asking and generating reports
→ Default: Editable

Step 4: Start Research
→ Team members can immediately see this research
→ Can view progress in real-time
```

---

**Share Existing Research Later**:

```
If research was created as private, can share later:

Step 1: Open private research
Step 2: Click "Share" button in top right
Step 3: Select sharing scope and permissions:
- Share with: Entire team / Specific members
- Permissions: View only / Editable
Step 4: Confirm sharing

After sharing:
✅ Selected members can access the research
✅ Shows in their "My Research"
✅ Marked as "Shared"
```

### Permission Management

#### 3 Permission Levels

**View Only**:
```
Can:
✅ View research content
✅ View conversation history
✅ View reports
✅ Export reports

Cannot:
❌ Cannot ask AI
❌ Cannot edit research
❌ Cannot generate new reports
❌ Cannot delete research

Suitable For:
- Executives viewing research results
- External consultants viewing temporarily
- Archived historical research
```

---

**Editable**:
```
Can:
✅ All "View Only" permissions
✅ Ask AI
✅ Generate new reports
✅ Add comments

Cannot:
❌ Cannot delete research (only creator and admin)
❌ Cannot modify research settings

Suitable For:
- Team collaborative research
- Sharing between product managers
- Continuously digging insights
```

---

**Administrator**:
```
Can:
✅ All "Editable" permissions
✅ Delete research
✅ Modify research settings
✅ Transfer ownership

Suitable For:
- Team administrators
- Department heads
```

### Q1: Can Personal Version Share?

**No, only Team version supports sharing.**

**Version Comparison**:
```
Free/Professional Version:
❌ No persona library sharing

❌ No Sage expert sharing

Team Version:

---

### Q2: Can Sharing Be Revoked?

**Yes, can revoke anytime.**

**Operations**:
```
Step 1: Open shared research/persona
Step 2: Click "Sharing Settings"
Step 3: Remove specific members or make private
Step 4: Confirm

After Revocation:
✅ Removed members immediately lose access
✅ Research/persona invisible to them
```

---

### Q3: Can Shared Research Be Password Protected?

**No password protection, but permission control available.**

**Permission Control**:
```
More secure approach:
✅ Share only with specific members (not entire team)
✅ Set "View Only" permission (cannot edit)
✅ Regularly review sharing settings
✅ Keep sensitive research private

Not supported:
❌ Password protection
❌ External link sharing (team only)
```

---

### Q4: Does Sharing Personas Consume My Quota?

**No, team shares quota.**

**Team Version Quota**:
```
Team Version:
✅ Unlimited Interview/Scout/Reports
✅ Entire team shares quota (not allocated per person)
✅ No matter who uses, deducted from team quota

Example:
- Team has 5 members
- Li Na does 10 Interviews with team personas
- Zhang Wei does 15 Interviews with team personas
- Both deducted from team quota, not separately billed
```

---

### Q5: Does Deleting Team Personas Affect Historical Research?

**No impact on historical research, but cannot create new research using it.**

**After Deletion**:
```
Historical Research:
✅ Research using that persona retains complete records
✅ Conversation history unaffected
✅ Can continue viewing

New Research:
❌ Cannot select that persona
❌ Cannot create new interviews using that persona

Recommendation:
- Don't casually delete widely used personas
- Check usage count before deletion
- Consider "archiving" instead of deleting
```

---

### Q6: Can Share with External Partners?

**No external sharing, team only.**

**Current Limitations**:
```
Can only share with:
✅ Team members (colleagues who joined the team)

Cannot share with:
❌ External partners
❌ Clients
❌ Consultants (unless invited to join team)
```

**Alternative Solution**:
```
If need to share externally:
1. Export report (Markdown/PDF/Word)
2. Share via email or other methods
3. External people view exported report

May support in future:
- External link sharing (roadmap)
- Password-protected sharing
```

### Q7: How Long Do Sharing Settings Last?

**Permanent, until you modify or delete.**

**Retention Mechanism**:
```
Sharing Settings:
✅ Permanently effective
✅ No automatic expiration
✅ Not automatically adjusted when members leave

If need to modify:
- Manually enter sharing settings

```
