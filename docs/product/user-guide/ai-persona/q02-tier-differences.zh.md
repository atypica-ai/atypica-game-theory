# 怎么选择合适的AI人设？

## 问题类型
用户手册问题

---

## 快速回答

**atypica提供两大类AI人设：公众人设库（30万+）和自定义人设。**

**推荐选择逻辑**：
- ✅ **常规研究**：直接使用公众人设库（快速、覆盖面广）
- ✅ **特定需求**：创建自定义人设（上传访谈数据或使用社交媒体观察）

**简单说**：大多数情况下，公众人设库就够用了；只有特别需要模拟特定真实用户时，才需要自定义创建。

---

## 详细说明

### 公众人设库：30万+预构建人设

**特点**：
- **数量**：30万+覆盖多样化人群
- **来源**：
  - ✅ 基于真实的研究背景构建（每个人设对应1个真实研究问题）
  - ✅ 基于真实社交媒体观察构建（每个人设3000字观察记录）
- **质量**：
  - ✅ 一致性达到80分（接近真人基准81分）
  - ✅ 质量稳定，经过验证
- **可见性**：所有用户共享（免费版和付费版都能用）

**适用场景**：
```
✅ 产品概念测试
✅ 市场趋势探索
✅ 用户态度调研
✅ 创意激发讨论
✅ 快速验证假设
✅ 大规模人群研究
```

**案例**：
- 测试气泡咖啡新品定价（选择10-15个目标人群）
- 了解健身App用户需求（选择运动爱好者人设）
- 探索Z世代消费偏好（筛选18-25岁人设）

**优势**：
- ✅ 立即可用，无需创建
- ✅ 覆盖面广，几乎任何人群都有
- ✅ 质量稳定，已经过验证
- ✅ 成本低，无额外创建成本

---

### 自定义人设：为特定需求定制

**特点**：
- **数量**：根据你的需求创建
- **来源**：基于你提供的数据（访谈记录、社交媒体观察）
- **质量**：取决于你提供的数据质量
- **可见性**：完全私有，只有你能用（团队版可共享给团队）

**适用场景**：
```
✅ 模拟特定真实用户
✅ 企业客户研究
✅ 敏感数据研究
✅ 公众库找不到的小众人群
✅ 需要长期追踪的固定用户群
✅ 团队共享的核心人设
```

**案例**：
- 导入500个会员访谈数据，测试新产品（模拟真实会员）
- 观察小红书上的"露营爱好者"，创建人设（公众库没有）
- 持续追踪固定用户群的需求变化（多次复用）

**创建方式**：
1. **上传访谈记录**：上传真人访谈的文字或音频，AI自动生成人设
2. **社交媒体观察**：让AI观察社交媒体上的目标人群，生成人设

**优势**：
- ✅ 完全定制，符合你的特定需求
- ✅ 数据私有，保护敏感信息
- ✅ 可复用，形成公司的"用户资产"
- ✅ 团队共享，提升协作效率

## 决策框架：我应该选哪个？

### 场景1：常规产品测试

```
需求：测试新产品概念

推荐：公众人设库
原因：
→ 30万+库覆盖大多数目标人群
→ 立即可用，节省时间
→ 质量稳定，结果可靠

操作：
Step 1: 描述目标人群（如"25-35岁女性，喜欢健康饮食"）
Step 2: 系统推荐10-15个匹配人设
Step 3: 选择人设，开始访谈
```

---

### 场景2：特定人群（公众库找不到）

```
需求：研究"冲浪爱好者"（小众人群）

推荐：自定义人设（社交媒体观察）
原因：
→ 公众库可能没有或覆盖不足
→ Scout可以观察小红书、抖音上的真实冲浪爱好者
→ 生成更符合真实群体的人设

操作：
Step 1: 使用社交媒体观察功能
Step 2: 指定观察目标"小红书上的冲浪爱好者"
Step 3: AI观察后生成人设
Step 4: 保存到你的人设库，供后续使用
```

---

### 场景3：模拟真实客户

```
需求：测试新功能，想模拟真实VIP客户

推荐：自定义人设（上传访谈记录）
原因：
→ 需要模拟特定真实用户
→ 你有历史访谈数据
→ 需要保护客户隐私

操作：
Step 1: 上传VIP客户访谈记录（匿名化）
Step 2: AI生成人设
Step 3: 使用人设测试新功能
Step 4: 后续可重复使用，无需再次访谈真实客户
```

---

### 场景4：团队协作研究

```
需求：产品团队需要共享核心用户人设

推荐：自定义人设（团队版）
原因：
→ 团队需要统一的用户理解
→ 多次研究复用同一批人设
→ 形成公司的"用户资产"

操作：
Step 1: 创建自定义人设（基于真实用户数据）
Step 2: 保存到团队人设库
Step 3: 团队成员都能使用
Step 4: 持续积累，越用越有价值
```

## 常见问题

### Q1：公众人设库的质量可靠吗？

**可靠，接近真人表现。**

```
质量保证：
✅ 基于真实的研究背景构建（30万+真实研究问题）
✅ 基于真实社交媒体观察构建（每个人设3000字观察记录）
✅ 一致性达到80分（接近真人基准81分）
✅ 30万+库，覆盖多样化人群
✅ 持续更新和优化

适用范围：
→ 90%的常规研究都够用
→ 产品测试、市场探索、用户调研等
```

---

### Q2：什么时候需要创建自定义人设？

**只有特定需求时才需要。**

```
需要自定义的情况：
✅ 公众库找不到匹配人群（小众、特殊）
✅ 需要模拟特定真实用户（你的客户）
✅ 有敏感数据，需要私有存储
✅ 团队需要长期复用的核心人设

不需要自定义的情况：
→ 常见人群（25-45岁职场人士、大学生、宝妈等）
→ 常规研究（产品测试、市场调研）
→ 快速验证假设
```

---

### Q3：自定义人设创建成本高吗？

**取决于你的数据和使用频次。**

```
创建成本：
- 上传访谈记录：需要有访谈数据（可能已有）
- 社交媒体观察：需要消耗Scout额度

但价值在于：
✅ 一次创建，永久保存
✅ 可用于多个研究项目
✅ 边际成本极低（越用越划算）
✅ 可分享给团队（团队版）
✅ 形成公司的"用户资产"

适合场景：
→ 核心目标人群（会长期研究）
→ 团队协作（多人使用）
→ 持续产品迭代（多次测试）

不适合场景：
→ 一次性研究
→ 非核心人群
→ 公众库已有合适人设
```

---

### Q4：自定义人设会被其他用户看到吗？

**不会，完全私有。**

```
隐私保护：
✅ 自定义人设只有创建者可见
✅ 其他用户无法搜索或使用
✅ 数据完全隔离
✅ 团队版可选择性共享给团队成员

vs 公众人设库：
→ 公众人设：所有用户共享
→ 自定义人设：完全私有
```

### Q6：可以把公众人设"收藏"到我的库吗？

**可以，使用"我的人设库"功能。**

```
操作：
Step 1: 从公众库中找到合适的人设
Step 2: 点击"添加到我的人设库"
Step 3: 该人设出现在你的个人库中
Step 4: 后续使用更方便，无需重复搜索

价值：
→ 快速访问常用人设
→ 整理和分类你的人设
→ 团队可共享收藏的人设
```
