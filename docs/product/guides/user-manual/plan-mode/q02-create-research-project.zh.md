# 如何创建我的第一个研究项目？

## 问题类型
用户手册问题

---

## 快速开始（5分钟）

### Step 1：进入创建页面

**入口**：
- 首页 → "开始新研究"
- 或侧边栏 → "创建研究"

---

### Step 2：描述研究需求

**输入框**："你想研究什么？"

**示例输入**：
```
"我想测试气泡咖啡新品的接受度"
```

**提示**：
- 用自然语言描述
- 1-2句话即可
- 不需要很专业

**点击"开始"**

---

### Step 3：Plan Mode对话澄清

**AI会问你一些问题**，帮助明确研究意图。

---

### Step 4：审核研究计划

**AI生成完整计划**，你审核并确认。

---

### Step 5：开始执行

**点击"确认开始"**，AI自动执行研究。

---

## 详细创建流程

### Phase 1：描述需求（1分钟）

#### 输入内容示例

**产品测试**：
```
"测试我的新产品：0糖气泡咖啡，
目标用户是25-35岁白领女性"
```

**用户洞察**：
```
"了解用户为什么流失，
我们是一个健身App"
```

**市场分析**：
```
"分析Z世代对可持续时尚的态度和支付意愿"
```

---

#### 输入技巧

**✅ 好的输入**：
- 明确具体：说清楚研究对象和目标
- 包含背景：简单介绍你的产品/行业
- 说明限制：如果有预算或时间限制

**❌ 避免的输入**：
- 太模糊："帮我做市场调研"
- 太复杂："分析整个行业的未来趋势和我们的战略定位以及..."
- 太简单："咖啡"（AI不知道你想研究什么）

---

### Phase 2：Plan Mode澄清（3-5轮对话）

#### AI会问的典型问题

**问题1："你的目标用户是谁？"**

**回答示例**：
- ✅ "25-35岁一线城市白领女性，经常喝咖啡"
- ✅ "不确定，帮我分析"
- ❌ "所有人"（太宽泛）

---

**问题2："你想了解哪些方面？"**

**回答示例**：
- ✅ "接受度、定价、包装设计"
- ✅ "全面了解，你推荐"
- ❌ "什么都想知道"（需要具体化）

---

**问题3："有什么特殊要求吗？"**

**回答示例**：
- ✅ "预算有限，希望2小时内完成"
- ✅ "需要向老板汇报，要用SWOT框架"
- ✅ "没有特殊要求"

---

**问题4："有参考材料吗？"**

**回答示例**：
- ✅ "有产品介绍PPT"（上传文件）
- ✅ "有竞品网站链接"
- ✅ "没有"

---

#### 澄清过程示例

**对话流程**：
```
用户："测试气泡咖啡新品"

AI："好的，你的目标用户是谁？"

用户："25-35岁白领女性"

AI："你想了解哪些方面？接受度、定价还是其他？"

用户："接受度和定价"

AI："有参考材料吗？比如产品介绍或包装设计稿？"

用户："有产品介绍PPT"（上传文件）

AI："明白了。我会设计一个研究计划，
包括用户讨论和深度访谈来测试接受度和定价。
稍等片刻..."
```

---

### Phase 3：审核研究计划（2-5分钟）

#### AI生成的计划示例

```
研究计划：气泡咖啡新品测试

研究目标：
- 测试产品接受度
- 验证定价策略（¥15-20区间）
- 收集用户反馈

目标用户画像：
- 25-35岁一线城市白领女性
- 经常喝咖啡（每周3次以上）
- 关注健康，愿意尝试新品

研究步骤：

1. 定义目标用户（10分钟）
   - 从人设库筛选8个符合条件的AI人设
   - 2个价格敏感型
   - 3个健康焦虑型
   - 3个社交尝鲜型

2. 群体讨论（30-40分钟）
   - 8人Discussion
   - 讨论产品概念和定价
   - 快速识别接受度和争议点

3. 深度访谈（60-80分钟）
   - 选择5人进行1v1 Interview
   - 深度挖掘购买动机
   - 测试3个定价方案（¥15/¥18/¥20）

4. 生成研究报告（10分钟）
   - 接受度评估
   - 定价建议
   - 产品优化方向

预计时间：2-3小时
预计成本：Token约¥80-120

输出：
- 完整研究报告（PDF）
- 用户访谈记录
- 可执行的产品和定价建议
```

---

#### 审核要点

**检查1：研究目标对吗？**
- 是否符合你的真实需求？
- 有没有遗漏重要方面？

**检查2：目标用户对吗？**
- 人群定义是否准确？
- 筛选条件是否合理？

**检查3：研究方法合适吗？**
- Discussion + Interview组合是否合适？
- 人数（8人讨论 + 5人访谈）是否够？

**检查4：时间和成本可接受吗？**
- 预计时间：2-3小时
- 预计成本：¥80-120
- 是否在预算内？

---

#### 你的选择

**选项1：满意，直接开始**
- 点击"确认开始"
- AI立即执行

**选项2：需要调整**
- 点击"修改计划"
- 告诉AI具体改哪里
- AI更新计划

**选项3：不满意，重新生成**
- 点击"重新生成"
- AI基于需求生成新计划

---

### Phase 4：执行研究（自动）

**AI开始执行**：
- 按照计划逐步执行
- 实时显示进度
- 可中途查看进展

**你可以**：
- 继续使用其他功能
- 关闭页面（研究继续）
- 等待完成通知

## 高级技巧

### 技巧1：上传参考材料

**支持的材料**：
- 产品介绍PPT
- 竞品链接
- 用户反馈截图
- 访谈记录（如果有）

**好处**：
- AI更理解你的产品
- 生成的计划更精准
- 节省澄清时间

**示例**：
```
用户："测试新产品概念"
AI："能分享产品介绍吗？"
用户：上传产品介绍PPT
AI：阅读后生成更精准的研究计划
```

---

### 技巧2：明确预算和时间限制

**提前说明限制**：
```
用户："我想做全面市场分析，
但预算有限，希望3小时内完成"

AI："明白了。我会设计一个精简版的计划，
优先做最重要的部分"
```

**AI会调整**：
- 减少访谈人数
- 简化研究步骤
- 聚焦核心问题

---

### 技巧3：指定研究框架

**如果你有偏好**：
```
用户："用SWOT框架分析我们的竞争优势"

AI："好的，我会用SWOT框架设计研究计划"
```

**AI会按照你指定的框架**：
- 设计研究步骤
- 组织报告结构
- 确保输出符合框架

---

### 技巧4：参考之前的研究

**如果不是第一次**：
```
用户："和上次的研究类似，
但这次测试不同的产品"

AI："明白了。我会参考你上次的研究结构，
调整为新产品"
```

**好处**：
- 保持研究一致性
- 结果可对比
- 节省设计时间

## 常见问题

### Q1：创建研究需要多久？

**时间分配**：
- 输入需求：1分钟
- Plan Mode澄清：3-5分钟
- 审核计划：2-5分钟
- **总计：5-10分钟**

---

### Q2：必须上传参考材料吗？

**不是必须**。
- 有材料：AI理解更准确
- 没材料：AI会基于描述推断

**建议**：
- 如果有，最好上传
- 如果没有，也可以继续

---

### Q3：Plan Mode澄清会问很多问题吗？

**通常3-5个问题**：
- 目标用户
- 研究范围
- 特殊要求
- 参考材料

**如果需求明确**：
- 可能只问1-2个问题

**如果需求模糊**：
- 可能问5-8个问题

---

### Q4：可以跳过Plan Mode直接执行吗？

**不建议**：
- Plan Mode帮助明确需求
- 避免方向错误
- 节省执行时间

**但如果你是熟练用户**：
- 可以提供非常详细的需求
- Plan Mode会快速生成计划
- 审核后立即执行

---

### Q5：计划生成后还能修改吗？

**可以**。你可以：
- 修改研究步骤
- 调整目标用户
- 改变研究方法
- 增删内容

**修改方式**：
- 点击"修改计划"
- 告诉AI具体改什么
- AI自动更新

## 实际案例

### 案例1：产品测试（最常见）

**用户输入**：
```
"测试我的新产品：一款智能手环，
主打运动监测和睡眠分析，
目标用户是25-40岁健身爱好者"
```

**Plan Mode生成的计划**：
```
研究计划：智能手环产品测试

步骤：
1. 筛选10个健身爱好者AI人设
2. Discussion讨论产品概念（8人）
3. Interview深度访谈（5人）
4. 测试定价（¥199/¥299/¥399）
5. 生成报告

预计时间：3小时
```

---

### 案例2：用户流失分析

**用户输入**：
```
"分析用户流失原因，
我们是一个在线教育平台，
用户在使用3个月后开始大量流失"
```

**Plan Mode生成的计划**：
```
研究计划：用户流失原因分析

步骤：
1. 定义流失用户画像
2. Interview深度访谈10个流失用户
3. 挖掘流失的真实原因
4. 对比活跃用户（5人）找差异
5. 生成流失分析报告 + 挽回策略

预计时间：4小时
```

---

### 案例3：市场机会分析

**用户输入**：
```
"我们想进入老年健身市场，
想知道市场机会和用户需求"
```

**Plan Mode生成的计划**：
```
研究计划：老年健身市场机会分析

步骤：
1. 市场趋势分析（webSearch）
2. 社交媒体观察老年健身社交媒体讨论
3. Interview访谈10个老年用户
4. 分析需求、痛点、支付意愿
5. 生成市场机会报告

预计时间：6小时
```
