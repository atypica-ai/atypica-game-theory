# How to Invite Colleagues to Use Together?

## Question Type
User Manual Question

---

## Quick Answer

**Need to upgrade to Team version to invite colleagues.**

**Operation Process**:
```
Step 1: Upgrade to Team version
→ Go to Account Settings → Upgrade to Team version
→ ¥1,999/month, up to 20 people

Step 2: Invite colleagues
→ Go to Team Settings → Click "Invite Members"
→ Enter colleague's email → Send invitation

Step 3: Colleague accepts invitation
→ Colleague receives email → Click link
→ Register/Login → Join team

Step 4: Start collaborating
→ Share persona library, research projects, Sage experts
```

**Version Comparison**:
- **Free/Professional version**: ❌ Doesn't support inviting colleagues (personal account)
- **Team version**: ✅ Supports inviting up to 20 people, team collaboration

### Invite Colleagues

#### Invitation Process

**Step 1: Go to Team Settings**

```
Path 1:
→ Account Settings → Team Management → Invite Members

Path 2:
→ Sidebar "Team" → Click "Invite Members" button
```

---

**Step 2: Enter Invitation Information**

```
Invitation form:
1. Invitation email:
   - Enter colleague's work email
   - Can invite multiple at once (one email per line)

2. Set role (optional):
   - Admin: Can manage team, invite/remove members
   - Member: Can use all features
   - Viewer: Can only view, not edit

3. Add welcome message (optional):
   - Custom welcome text
   - Explain why inviting to join

Example:
Invitation emails:
zhang.wei@example.com
li.na@example.com

Role: Member

Welcome message:
"Welcome to our product team!
 This is our tool for market research and user insights."
```

---

**Step 3: Send Invitation**

```
Click "Send Invitation" button

System operations:
✅ Send invitation email to colleague's inbox
✅ Show "Pending" status in team settings
✅ Generate unique invitation link (72 hours valid)
```

---

**Step 4: Colleague Accepts Invitation**

**Email colleague receives**:
```
Email subject:
[Your name] invites you to join [Team name] - atypica.AI

Email content:
Hello!

[Your name] invites you to join [Team name]'s atypica team.

atypica is an AI-driven market research tool, helps teams quickly understand user needs.

Team size: 5 people
Your role: Member

Welcome message:
"Welcome to our product team!..."

→ [Accept Invitation] button

This invitation link valid for 72 hours.
```

---

**Colleague operations**:
```
Step 1: Click "Accept Invitation" button in email

Step 2: If already has atypica account:
→ Log in to existing account
→ Confirm joining team

Step 3: If doesn't have atypica account:
→ Fill in registration info (username, password)
→ Automatically join team

Step 4: Join successful
→ See team welcome page
→ Can access team shared persona library, research projects
→ Start using
```

#### Modify Member Permissions

**Permission Levels**:

**Admin**:
```
Permissions:
✅ All member permissions
✅ Invite/remove members
✅ Modify member roles
✅ Manage team settings
✅ View billing and usage stats
✅ Delete team data

Applicable scenarios:
- Team leader
- Department manager
```

**Member**:
```
Permissions:
✅ Create and edit research projects
✅ Use team persona library
✅ Use team Sage experts
✅ Share research results
❌ Can't invite/remove other members
❌ Can't modify team settings

Applicable scenarios:
- Product managers
- Market researchers
- Designers
```

**Viewer**:
```
Permissions:
✅ View team research results
✅ View team persona library (read-only)
✅ View team Sage experts
❌ Can't create or edit
❌ Can't use AI features (Interview, Scout)

Applicable scenarios:
- Executives (only need to see results)
- External consultants (temporary view)
- Interns (learning phase)
```

---

**Modify Permissions**:
```
Step 1: Go to Team Settings → Member Management
Step 2: Find member to modify
Step 3: Click "Edit" button
Step 4: Select new role
Step 5: Confirm modification

Example:
Promote "Li Na" from Member to Admin:
→ Click "Edit"
→ Select "Admin"
→ Confirm
→ Li Na immediately gets admin permissions
```

---

#### Remove Members

**Removal Process**:
```
Step 1: Go to Team Settings → Member Management
Step 2: Find member to remove
Step 3: Click "Remove" button
Step 4: Confirm removal

Confirmation dialog:
"Confirm remove [member name]?
 After removal:
 - Member will lose team data access
 - Member's personal research can be retained or deleted
 - This operation cannot be undone"

Step 5: Select data handling:
□ Retain member's personal research (transfer to team)
□ Delete member's all personal research

Step 6: Confirm removal
```

---

**Removal Consequences**:
```
Removed member:
❌ Immediately loses team access
❌ Can't access team persona library
❌ Can't access team research projects
❌ Can't use team Sage experts
✅ Can choose to keep personal account (downgrade to free version)
✅ Or completely delete account
```

**No, only Team version supports.**

**Reason**:
- Free/Professional version: Designed for individual use
- Team version: Specifically designed for team collaboration

**If need to invite colleagues**:
- Upgrade to Team version: ¥1,999/month
- Or each person independently purchases Professional version: ¥499/month (but can't collaborate)

**Cost Comparison**:
```
5-person team:
Plan A (each buys Professional version):
- Cost: ¥499 × 5 = ¥2,495/month
- Problem: Can't collaborate

Plan B (Team version):
- Cost: ¥1,999/month
- Advantage: Collaboration + save ¥496/month
```

### Q4: What if colleague rejects invitation?

**Resend invitation.**

**Operations**:
```
Step 1: Go to Team Settings → Member Management
Step 2: Find "Pending" status member
Step 3: Click "Resend Invitation"
Step 4: New invitation email sent

If continues rejecting:
- Ask colleague reason
- Or remove invitation
```

---

### Q5: How to handle after member leaves?

**Remove member, data can be retained or deleted.**

**Operation Process**:
```
Step 1: Go to Team Settings → Member Management
Step 2: Find departed member
Step 3: Click "Remove"
Step 4: Select data handling:
- Retain data (transfer to team)
- Delete data (completely remove)
Step 5: Confirm removal

After removal:
✅ Member immediately loses access
✅ Team data won't be taken away
✅ Meets enterprise data security requirements
```

---

### Q6: Can different members have different permissions?

**Yes, supports 3 roles.**

**Role Permissions**:
```
Admin:
✅ Full control (invite/remove/manage)

Member:
✅ Create and edit research
❌ Can't manage team

Viewer:
✅ Can only view
❌ Can't edit or create
```

---

### Q7: Can Team version downgrade to personal version?

**Yes, but note data handling.**

**Downgrade Process**:
```
Step 1: Go to Team Settings
Step 2: Click "Downgrade to Personal Version"
Step 3: Select data handling:
- Retain your personal data
- Other members' data transfer to other admin or delete
Step 4: Confirm downgrade

After downgrade:
- You become Professional version (¥499/month)
- Other members lose access
- Team data retained or deleted based on choice
```
