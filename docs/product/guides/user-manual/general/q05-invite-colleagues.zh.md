# 如何邀请同事一起使用？

## 问题类型
用户手册问题

---

## 快速回答

**需要升级到团队版才能邀请同事。**

**操作流程**：
```
Step 1: 升级到团队版
→ 进入账号设置 → 升级到团队版
→ ¥1,999/月，最多20人

Step 2: 邀请同事
→ 进入团队设置 → 点击「邀请成员」
→ 输入同事邮箱 → 发送邀请

Step 3: 同事接受邀请
→ 同事收到邮件 → 点击链接
→ 注册/登录 → 加入团队

Step 4: 开始协作
→ 共享人设库、研究项目、Sage专家
```

**版本对比**：
- **免费版/专业版**：❌ 不支持邀请同事（个人账号）
- **团队版**：✅ 支持邀请最多20人，团队协作

## 详细说明

### 团队版 vs 个人版

#### 为什么需要团队版？

**个人版（免费/专业）**：
```
设计目的：个人独立使用
限制：
❌ 无法邀请同事
❌ 无法共享人设库
❌ 无法共享研究项目
❌ 每人独立购买和使用
```

**团队版**：
```
设计目的：团队协作
功能：
✅ 邀请最多20个同事
✅ 共享团队人设库
✅ 共享研究项目
✅ 协同编辑和讨论
✅ 统一账单和管理
```

---

#### 升级到团队版

**升级流程**：

```
Step 1: 进入账号设置
→ 点击右上角头像
→ 选择「账号设置」

Step 2: 点击「升级到团队版」
→ 查看团队版功能和价格
→ ¥1,999/月 或 ¥19,188/年（8折）
→ 最多20个成员

Step 3: 选择付款方式
→ 月付：¥1,999/月
→ 年付：¥19,188/年（节省¥4,800）
→ 支付方式：支付宝、微信、企业转账

Step 4: 确认升级
→ 填写团队信息：
  - 团队名称（如：XX公司产品团队）
  - 初始成员数
→ 确认支付

Step 5: 升级完成
→ 立即生效
→ 个人数据自动转为团队共享
→ 可以开始邀请成员
```

### 邀请同事

#### 邀请流程

**Step 1: 进入团队设置**

```
路径1:
→ 账号设置 → 团队管理 → 邀请成员

路径2:
→ 侧边栏「团队」→ 点击「邀请成员」按钮
```

---

**Step 2: 输入邀请信息**

```
邀请表单：
1. 邀请邮箱：
   - 输入同事的工作邮箱
   - 可以一次邀请多人（每行一个邮箱）

2. 设置角色（可选）：
   - 管理员：可以管理团队、邀请/移除成员
   - 成员：可以使用所有功能
   - 查看者：只能查看，不能编辑

3. 添加欢迎消息（可选）：
   - 自定义欢迎文字
   - 说明为什么邀请加入

示例：
邀请邮箱：
zhang.wei@example.com
li.na@example.com

角色：成员

欢迎消息：
「欢迎加入我们的产品团队！
 这是我们用于市场研究和用户洞察的协作工具。」
```

---

**Step 3: 发送邀请**

```
点击「发送邀请」按钮

系统操作：
✅ 发送邀请邮件到同事邮箱
✅ 在团队设置中显示「待接受」状态
✅ 生成唯一的邀请链接（72小时有效）
```

---

**Step 4: 同事接受邀请**

**同事收到的邮件**：
```
邮件标题：
[你的名字] 邀请你加入 [团队名称] - atypica.AI

邮件内容：
你好！

[你的名字] 邀请你加入 [团队名称] 的atypica团队。

atypica是AI驱动的市场研究工具，帮助团队快速了解用户需求。

团队人数：5人
你的角色：成员

欢迎消息：
「欢迎加入我们的产品团队！...」

→ [接受邀请] 按钮

此邀请链接72小时内有效。
```

---

**同事操作**：
```
Step 1: 点击邮件中的「接受邀请」按钮

Step 2: 如果已有atypica账号：
→ 登录现有账号
→ 确认加入团队

Step 3: 如果没有atypica账号：
→ 填写注册信息（用户名、密码）
→ 自动加入团队

Step 4: 加入成功
→ 看到团队欢迎页面
→ 可以访问团队共享的人设库、研究项目
→ 开始使用
```

### 成员管理

#### 查看团队成员

**成员列表**：
```
位置：团队设置 → 成员管理

显示内容：
1. 成员头像和名字
2. 邮箱
3. 角色（管理员/成员/查看者）
4. 加入时间
5. 最后活跃时间
6. 操作按钮（编辑/移除）

示例：
张伟  zhang.wei@example.com  管理员  2026-01-15  2分钟前
李娜  li.na@example.com      成员    2026-01-20  1小时前
王芳  wang.fang@example.com  查看者  2026-01-25  昨天
```

---

#### 修改成员权限

**权限级别**：

**管理员**：
```
权限：
✅ 所有成员权限
✅ 邀请/移除成员
✅ 修改成员角色
✅ 管理团队设置
✅ 查看账单和使用统计
✅ 删除团队数据

适用场景：
- 团队负责人
- 部门经理
```

**成员**：
```
权限：
✅ 创建和编辑研究项目
✅ 使用团队人设库
✅ 使用团队Sage专家
✅ 共享研究成果
❌ 不能邀请/移除其他成员
❌ 不能修改团队设置

适用场景：
- 产品经理
- 市场研究人员
- 设计师
```

**查看者**：
```
权限：
✅ 查看团队研究成果
✅ 查看团队人设库（只读）
✅ 查看团队Sage专家
❌ 不能创建或编辑
❌ 不能使用AI功能（Interview、Scout）

适用场景：
- 高管（只需看结果）
- 外部顾问（临时查看）
- 实习生（学习阶段）
```

---

**修改权限**：
```
Step 1: 进入团队设置 → 成员管理
Step 2: 找到要修改的成员
Step 3: 点击「编辑」按钮
Step 4: 选择新角色
Step 5: 确认修改

示例：
将「李娜」从成员提升为管理员：
→ 点击「编辑」
→ 选择「管理员」
→ 确认
→ 李娜立即获得管理员权限
```

---

#### 移除成员

**移除流程**：
```
Step 1: 进入团队设置 → 成员管理
Step 2: 找到要移除的成员
Step 3: 点击「移除」按钮
Step 4: 确认移除

确认对话框：
「确定要移除 [成员名] 吗？
 移除后：
 - 该成员将失去团队数据访问权限
 - 该成员的个人研究可以选择保留或删除
 - 此操作不可撤销」

Step 5: 选择数据处理方式：
□ 保留该成员的个人研究（转移给团队）
□ 删除该成员的所有个人研究

Step 6: 确认移除
```

---

**移除后果**：
```
被移除的成员：
❌ 立即失去团队访问权限
❌ 无法访问团队人设库
❌ 无法访问团队研究项目
❌ 无法使用团队Sage专家
✅ 可以选择保留个人账号（降为免费版）
✅ 或完全删除账号
```

### 团队协作功能

#### 1. 共享人设库

**团队人设库**：
```
功能：
✅ 所有成员共享同一个人设库
✅ 任何成员创建的人设，全团队可用
✅ 基于真人访谈的高质量自定义人设积累

价值：
- 新成员快速上手（使用团队已有人设）
- 知识资产积累（不会因为成员离职而丢失）
- 避免重复构建人设

使用方式：
→ 创建人设时，选择「团队人设库」
→ 该人设自动共享给全团队
→ 其他成员在人设选择器中可以看到
```

---

#### 2. 共享研究项目

**研究项目共享**：
```
功能：
✅ 选择性共享研究项目
✅ 团队成员可以查看/编辑（根据权限）
✅ 协同讨论和批注

共享方式：
→ 创建研究时，选择「团队可见」
→ 或在已有研究中点击「共享给团队」

权限设置：
- 仅查看：只能查看研究内容
- 可编辑：可以继续追问和生成报告
- 管理员：可以删除研究

使用场景：
- 产品经理分享用户需求研究
- 市场部分享市场洞察
- 设计师分享用户体验研究
```

---

#### 3. 共享Sage专家

**Sage专家共享**：
```
功能：
✅ 团队级别的Sage专家
✅ 所有成员都能咨询同一个专家
✅ 专家知识持续积累

团队Sage案例：
1. UX设计专家：基于公司设计规范
2. 产品策略专家：基于历史产品文档
3. 市场营销专家：基于营销案例库
4. 技术架构专家：基于技术文档

价值：
- 知识传承（老员工经验转化为Sage专家）
- 新人培训（新人直接咨询团队Sage专家）
- 统一标准（全团队基于相同专家建议）
```

## 常见问题

### Q1：免费版或专业版可以邀请同事吗？

**不可以，只有团队版支持。**

**原因**：
- 免费版/专业版：设计为个人使用
- 团队版：专门为团队协作设计

**如果需要邀请同事**：
- 升级到团队版：¥1,999/月
- 或每人独立购买专业版：¥499/月（但无法协作）

**成本对比**：
```
5人团队：
方案A（每人买专业版）：
- 成本：¥499 × 5 = ¥2,495/月
- 问题：无法协作

方案B（团队版）：
- 成本：¥1,999/月
- 优势：协作 + 节省¥496/月
```

---

### Q2：可以邀请多少人？

**团队版标准：最多20人。**

**超过20人**：
- 联系销售团队
- 定制企业方案
- 邮件：enterprise@atypica.ai

---

### Q3：被邀请的同事需要单独付费吗？

**不需要，团队版费用已包含所有成员。**

**计费方式**：
```
团队版：¥1,999/月（最多20人）
- 不管实际有几个成员
- 不按成员数额外收费
- 5个人和20个人，费用相同
```

---

### Q4：同事拒绝邀请怎么办？

**重新发送邀请。**

**操作**：
```
Step 1: 进入团队设置 → 成员管理
Step 2: 找到「待接受」状态的成员
Step 3: 点击「重新发送邀请」
Step 4: 新邀请邮件发送

如果持续拒绝：
- 询问同事原因
- 或移除该邀请
```

---

### Q5：成员离职后如何处理？

**移除成员，数据可选择保留或删除。**

**操作流程**：
```
Step 1: 进入团队设置 → 成员管理
Step 2: 找到离职成员
Step 3: 点击「移除」
Step 4: 选择数据处理方式：
- 保留数据（转移给团队）
- 删除数据（彻底移除）
Step 5: 确认移除

移除后：
✅ 该成员立即失去访问权限
✅ 团队数据不会被带走
✅ 符合企业数据安全要求
```

---

### Q6：可以设置不同成员的不同权限吗？

**可以，支持3种角色。**

**角色权限**：
```
管理员：
✅ 完全控制（邀请/移除/管理）

成员：
✅ 创建和编辑研究
❌ 不能管理团队

查看者：
✅ 只能查看
❌ 不能编辑或创建
```

---

### Q7：团队版可以降级为个人版吗？

**可以，但数据处理需注意。**

**降级流程**：
```
Step 1: 进入团队设置
Step 2: 点击「降级为个人版」
Step 3: 选择数据处理：
- 保留你的个人数据
- 其他成员数据转移给其他管理员 或 删除
Step 4: 确认降级

降级后：
- 你变为专业版（¥499/月）
- 其他成员失去访问权限
- 团队数据根据选择保留或删除
```
