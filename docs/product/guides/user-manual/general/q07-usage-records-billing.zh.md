# 如何查看我的使用记录和账单？

## 问题类型
用户手册问题

---

## 快速回答

**在账号设置中查看使用记录和账单。**

**快速操作**：
```
Step 1: 点击右上角头像
Step 2: 选择「账号设置」
Step 3: 点击「使用统计」或「账单管理」

可以查看：
✅ 使用统计：Interview、Scout、报告、播客次数
✅ 额度余额：当前周期剩余额度
✅ 账单历史：所有付款记录
✅ 发票管理：下载和开具发票
```

## 常见问题

### Q1：使用记录会保留多久？

**永久保留。**

**保留内容**：
```
永久保留：
✅ 所有使用记录（Interview、Scout、报告）
✅ 所有账单记录
✅ 所有发票记录

价值：
- 随时回顾历史使用情况
- 财务审计和报销
- 年度使用分析
```

---

### Q2：额度每月什么时候重置？

**每月1日00:00重置（北京时间）。**

**重置规则**：
```
免费版/专业版：
- 每月1日00:00重置
- 上月未用完的额度不累积

团队版：
- 无限制，无需重置
- 播客额度每月1日重置（10次/月）

案例：
1月份用了3次Interview，剩余2次
2月1日：额度重置为5次（不是7次）
```

---

### Q3：可以导出使用记录吗？

**可以，支持CSV和PDF格式。**

**导出操作**：
```
Step 1: 进入使用统计
Step 2: 选择时间范围
- 本月
- 最近3个月
- 最近6个月
- 自定义时间段

Step 3: 选择导出格式
- CSV：适合Excel分析
- PDF：适合打印和归档

Step 4: 下载文件

文件内容：
- 日期时间
- 功能类型（Interview、Scout等）
- 项目名称
- 使用时长
- 消耗额度
```

---

### Q4：账单可以修改吗？

**不可以修改，但可以申请补开或重开。**

**如果需要修改**：
```
常见情况：
1. 发票抬头写错
   → 申请重开发票（原发票作废）

2. 金额错误
   → 联系客服核实并重开

3. 开票信息遗漏
   → 申请补开发票

操作：
→ 账单管理 → 找到账单
→ 点击「申请重开」
→ 填写正确信息
→ 提交申请
→ 客服审核并处理（1-3个工作日）
```

---

### Q5：团队成员可以查看团队账单吗？

**不可以，只有管理员可以。**

**权限划分**：
```
管理员：
✅ 查看团队账单
✅ 查看团队使用统计
✅ 管理支付方式
✅ 申请发票

普通成员：
✅ 查看自己的使用记录
❌ 不能查看团队账单
❌ 不能查看团队总使用量
❌ 不能查看其他成员的详细记录

查看者：
✅ 查看自己的使用记录
❌ 其他都不能查看
```

---

### Q6：忘记续费会怎样？

**账号会降级，但数据不会丢失。**

**忘记续费的后果**：
```
专业版 → 免费版：
❌ 失去无限制额度
✅ 恢复为5次Interview/月、2次Scout/月
✅ 所有历史数据保留
✅ 可以随时重新订阅恢复

团队版 → 专业版/免费版：
❌ 失去团队协作功能
❌ 成员失去访问权限
✅ 管理员数据保留
✅ 可以随时重新订阅恢复
```

**如何避免**：
```
建议：
✅ 开启自动续费（默认）
✅ 确保支付方式余额充足
✅ 绑定多个支付方式（备用）
✅ 开启邮件提醒（到期前7天提醒）
```

---

### Q7：可以申请退款吗？

**有退款政策，但有条件。**

**退款政策**：
```
7天无理由退款：
✅ 订阅7天内
✅ 使用少于3次Interview
✅ 全额退款

超过7天：
❌ 一般不支持退款
⚠️ 特殊情况可以协商（如系统bug导致无法使用）

申请流程：
Step 1: 账单管理 → 申请退款
Step 2: 填写退款原因
Step 3: 客服审核（1-3个工作日）
Step 4: 退款到原支付方式（5-7个工作日）
```

---

### Q8：发票可以补开吗？

**可以，1年内都可以补开。**

**补开流程**：
```
Step 1: 账单管理 → 发票管理
Step 2: 找到未开票的账单
Step 3: 点击「申请发票」
Step 4: 填写开票信息
Step 5: 提交申请
Step 6: 1-3个工作日开具

限制：
- 只能补开1年内的账单
- 超过1年无法补开
- 建议及时申请发票
```
