# How to Upload Interview Transcripts to Create AI Persona? What Formats Are Supported? How to Import My Existing User Data?

## Question Type
User Manual Question

---

## Supported Formats

### File Formats
- ✅ TXT (plain text)
- ✅ PDF
- ✅ Word (.doc / .docx)
- ✅ Markdown (.md)
- ✅ Excel / CSV (structured data)

### Content Requirements

**Ideal Quality (High-Quality Persona)**:
- In-depth interview transcript: 5000+ words
- Contains complete conversation process
- Covers 7 dimensions of information (demographics, values, consumption habits, lifestyle, decision logic, information channels, social relationships)

**Medium Quality (Usable Persona)**:
- Simple interview transcript: 2000-5000 words
- Or structured user data (CRM export)
- At least contains basic information and some behavioral characteristics

**Not Recommended**:
- Only basic information (name, age, gender)
- Lacks deep psychological and behavioral data

---

## Upload Steps

### Method 1: Upload Interview Transcripts

**Steps**:
1. Go to Persona management page
2. Click "Create Persona" → "Import Interview Transcripts"
3. Upload file (drag and drop or select file)
4. System automatically extracts information:
   - Basic information (age, gender, occupation, etc.)
   - Values and lifestyle
   - Consumption habits and decision logic
   - Pain points and needs
5. Preview and edit (optional)
6. Confirm and save to "My Persona Library"

**Processing Time**:
- 2000 words: About 1 minute
- 5000 words: About 2-3 minutes

---

### Method 2: Import Structured Data

**Use Case**: CRM export, user survey results

**Excel Template Example**:
```
| Name | Age | Gender | Occupation | City | Income | Values | Consumption Habits | Pain Points |
|------|-----|--------|------------|------|--------|--------|--------------------|-------------|
| Zhang Li | 28 | Female | PM | Shanghai | 15K | Value for money | Online shopping, read reviews | Time-pressed |
```

**Steps**:
1. Download Excel template
2. Fill in user data
3. Upload to atypica
4. System automatically converts to personas
5. Preview and confirm save

**Quality Tips**:
- System will prompt for missing key information
- Complete 7-dimension information provides higher quality personas
- Recommend filling in as detailed user information as possible

## Batch Import

### Supports Batch Creation

**Scenario**: Import multiple user data at once

**Method 1: Multiple Interview Transcripts**
- Compress into ZIP file
- Upload ZIP
- System automatically creates one persona per file

**Method 2: Excel Multiple Rows**
- Excel contains 50-500 rows of user data
- Upload all at once
- System batch generates 50-500 custom personas

**Limits**:
- Maximum 500 at once
- Total size not exceeding 100MB

## Best Practices

### Tips to Improve Quality

**Tip 1: Format Interview Transcripts**
```
Recommended format:
Q: What is your usual budget for buying coffee?
A: I usually buy ¥15-20, over ¥30 feels expensive.

Not recommended:
Interview subject indicated price sensitivity.
```
→ Preserving original dialogue provides higher quality

**Tip 2: Supplement Missing Dimensions**
- After upload, system prompts "missing dimensions"
- Can manually supplement
- Or conduct follow-up interview for AI to auto-supplement

**Tip 3: Regular Updates**
- User behavior changes
- Recommend updating every 6-12 months
- Or append when new interview data available

## Frequently Asked Questions

### Q1: What if uploaded interview transcripts contain sensitive information?

**Recommendation**:
1. Delete real names and contact information before upload
2. Or use pseudonyms
3. System will not leak original files

### Q2: Can I edit if I find errors after upload?

**Yes**:
1. Go to persona detail page
2. Click "Edit"
3. Modify 7-dimension information
4. Save updates

### Q3: Which fields are required for Excel import?

**Required**:
- Name (or ID)
- Age
- Gender
- Occupation
- City

**Optional but Recommended**:
- Income, Education (demographics)
- Values, Lifestyle (psychological)
- Consumption habits (behavioral)
- Pain points and needs
