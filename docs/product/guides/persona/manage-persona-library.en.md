# How to Manage and Organize My Persona Library?

## Question Type
User Manual Question

---

## Persona Library Structure

### Library Structure

```
My Persona Library
├── Public Library Personas (Read-only)
│   └── Used public personas (Can bookmark)
│
└── My Custom Personas (Editable)
    ├── Ungrouped
    ├── Project A
    ├── Project B
    └── Customer C
```

---

## Management Functions

### Function 1: Folder Organization

**Create Folders**:
1. Go to "My Personas"
2. Click "New Folder"
3. Name it (e.g. "2024Q1 Members", "VIP Customers")
4. Drag personas into folder

**Recommended Grouping Methods**:
- By project: Project A, Project B
- By time: 2024Q1, 2024Q2
- By user type: VIP, Regular members, Churned users
- By industry: Tech, Consumer, Healthcare

---

### Function 2: Tag Management

**Add Tags**:
- Select persona → Add tag
- Supports multiple tags: Price sensitive, Fitness enthusiast, Shanghai

**Filter by Tag**:
```
Filter: Tag = "Fitness enthusiast" + "Muscle building"
Result: Shows all personas matching criteria
```

**Recommended Tag System**:
- Demographics: Age range, Occupation, City
- Behavioral characteristics: Price sensitive, Quality-first, Early adopter
- Interest areas: Fitness, Beauty, Tech, Food

---

### Function 3: Bookmarks

**Bookmark Frequently Used Personas**:
- Public library personas you've used
- Click "Bookmark"
- Next time select directly from bookmarks, no need to search again

**Scenarios**:
- Certain research types frequently use same personas
- Establish "My Frequently Used Persona Library"

---

### Function 4: Filtering

**Filter by Persona Source**:
```
Filter: Source = Public Library
Result: Shows all personas bookmarked from public library

Filter: Source = Custom
Result: Shows all self-created personas
```

---

### Function 5: Batch Operations

**Supports Batch**:
- ✅ Batch move to folder
- ✅ Batch add tags
- ✅ Batch delete
- ✅ Batch export (Excel)

**Steps**:
1. Select multiple personas
2. Choose batch operation
3. Confirm execution

## Best Practices

### Recommendation 1: Regular Cleanup

**Clean Incomplete Personas**:
- Filter incomplete personas
- Delete or supplement information

**Clean Outdated Personas**:
- Personas unused for over 1 year
- Consider deleting or updating

---

### Recommendation 2: Build Domain Libraries

**Scenario**: Company frequently tests same product category

**Practice**:
1. Generate 20-30 high-quality personas in first research
2. Create folder "Fitness App Users"
3. Reuse directly for subsequent research
4. Regular updates (every 6-12 months)

**Advantages**:
- Research results comparable
- No need to regenerate each time
- Accumulate domain-specific persona library

## Export and Backup

### Export Persona Data

**Formats**:
- Excel: Contains 7-dimension detailed information
- JSON: Contains complete data structure

**Steps**:
1. Select personas
2. Click "Export"
3. Choose format
4. Download file

**Uses**:
- Backup persona data
- Share with team members
- Import to other tools

---

### Backup Recommendations

**Important Personas Recommend Backup**:
- Custom personas (self-created)
- High-value project personas
- Regular backups (monthly)

## Frequently Asked Questions

### Q1: Can public library personas be edited?

**Cannot edit directly**, but can:
1. Bookmark the persona
2. "Create copy based on this"
3. Edit copy, save as custom persona

### Q2: Can deleted personas be recovered?

**Can recover within 30 days**:
- Go to "Recycle Bin"
- Select persona
- Click "Recover"

**After 30 days**:
- Permanently deleted, cannot recover
- Recommend regular backups of important personas

### Q3: Is there a limit on persona library size?

**Current Limits**:
- Free version: Maximum 50 custom personas
- Paid version: Maximum 500 custom personas
- Enterprise version: Unlimited
