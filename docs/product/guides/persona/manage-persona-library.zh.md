# 怎么管理和整理我的人设库？

## 问题类型
用户手册问题

---

## 人设库结构

### 人设库结构

```
我的人设库
├── 公开库人设（只读）
│   └── 已使用的公众人设（可收藏）
│
└── 我的自定义人设（可编辑）
    ├── 未分组
    ├── 项目A
    ├── 项目B
    └── 客户C
```

---

## 管理功能

### 功能1：文件夹分组

**创建文件夹**：
1. 进入"我的Persona"
2. 点击"新建文件夹"
3. 命名（如"2024Q1会员"、"VIP客户"）
4. 拖拽人设到文件夹

**推荐分组方式**：
- 按项目：项目A、项目B
- 按时间：2024Q1、2024Q2
- 按用户类型：VIP、普通会员、流失用户
- 按行业：科技、消费、医疗

---

### 功能2：标签管理

**添加标签**：
- 选择人设 → 添加标签
- 支持多标签：价格敏感、健身爱好者、上海

**按标签筛选**：
```
筛选：标签 = "健身爱好者" + "增肌"
结果：显示所有符合条件的人设
```

**推荐标签体系**：
- 人口特征：年龄段、职业、城市
- 行为特征：价格敏感、品质优先、早期采用者
- 兴趣领域：健身、美妆、科技、美食

---

### 功能3：收藏夹

**收藏常用人设**：
- 从公开库使用过的人设
- 点击"收藏"
- 下次直接从收藏夹选择，无需重新搜索

**场景**：
- 某类研究经常用同一批人设
- 建立"我的常用人设库"

---

### 功能4：筛选功能

**按人设来源筛选**：
```
筛选：来源 = 公开库
结果：显示所有从公开库收藏的人设

筛选：来源 = 自定义
结果：显示所有自己创建的人设
```

---

### 功能5：批量操作

**支持批量**：
- ✅ 批量移动到文件夹
- ✅ 批量添加标签
- ✅ 批量删除
- ✅ 批量导出（Excel）

**操作**：
1. 勾选多个人设
2. 选择批量操作
3. 确认执行

## 最佳实践

### 建议1：定期清理

**清理信息不完整的人设**：
- 筛选信息不完整的人设
- 删除或补充信息

**清理过期人设**：
- 超过1年未使用的人设
- 考虑删除或更新

---

### 建议2：建立领域库

**场景**：公司经常测试同一品类产品

**做法**：
1. 第一次研究时生成20-30个高质量人设
2. 创建文件夹"健身App用户"
3. 后续研究直接复用
4. 定期更新（每6-12个月）

**优势**：
- 研究结果可对比
- 无需每次重新生成
- 积累领域专属人设库

## 导出和备份

### 导出人设数据

**格式**：
- Excel：包含7维度详细信息
- JSON：包含完整数据结构

**步骤**：
1. 选择人设
2. 点击"导出"
3. 选择格式
4. 下载文件

**用途**：
- 备份人设数据
- 分享给团队成员
- 导入到其他工具

---

### 备份建议

**重要人设建议备份**：
- 自定义人设（自己创建的）
- 高价值项目使用的人设
- 定期备份（每月）

## 常见问题

### Q1：公开库的人设可以编辑吗？

**不可以直接编辑**，但可以：
1. 收藏该人设
2. "基于此创建副本"
3. 编辑副本，保存为自定义人设

### Q2：删除人设后可以恢复吗？

**30天内可恢复**：
- 进入"回收站"
- 选择人设
- 点击"恢复"

**30天后**：
- 永久删除，无法恢复
- 建议定期备份重要人设

### Q3：人设库有数量限制吗？

**当前限制**：
- 免费版：最多50个自定义人设
- 付费版：最多500个自定义人设
- 企业版：无限制
