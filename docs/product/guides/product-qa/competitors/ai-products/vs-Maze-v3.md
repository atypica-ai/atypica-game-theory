## atypica's Unique Value

### 1. Validate Direction, Not Just Test Prototypes

**Maze's assumption**: Product direction decided, testing if prototype is usable.

**atypica's capability**:
- Validate demand before designing prototypes
- Avoid designing "usable but unwanted" features

**Case Study**:
Product team has 3 feature directions.

**Using Maze**:
- Build 3 prototypes (1 week)
- Test usability (3 days)
- Result: Feature C has best usability (90% completion rate)
- → Design Feature C

**Using atypica**:
```
Deep interviews revealed:
- Feature C: Great usability, but weak demand (3/10)
  "I rarely export data, why would I need this feature?"

- Feature A: Unknown usability, but strong demand (9/10)
  "This is exactly what I need! I'd pay 30% more."

→ Should build Feature A (strong demand)
→ Avoid wasting 2 weeks design + 4 weeks dev on Feature C
```

**The difference**: Maze tells you "which prototype works," atypica tells you "should you build this feature."

**Maze requires**: Existing prototype (Figma/Sketch) to test

**atypica can work with**:
- Just an idea
- No prototype needed
- No code needed

**Value scenario**:
- Test 50 product concepts, filter to most promising 5
- Maze: Need to build 50 prototypes (8-10 weeks)
- atypica: Directly interview and validate (3-5 days)

**Savings**: 8 weeks design time + avoid wrong directions

### 3. Understand Deep "Why"

**Maze's output**:
```
Task: Complete purchase
- Success rate: 65%
- Heatmap: 40% users click "Save" first (thinking it's purchase)
- Recommendation: Adjust button position and style
```

**atypica's output**:
```
Deep interviews revealed:
Not "can't find button," but "unsure whether to buy"

User A: "Button position doesn't matter. Key issue is no size recommendation.
       I don't know if it fits me, afraid to commit."

User B: "Not that I can't find the buy button, it's when I saw $15 shipping
       I immediately gave up. $30 item, shipping is half the price?"

→ Real issues: Lack of trust + pricing strategy
→ Optimization direction:
  1. Add size recommendations + user reviews
  2. Free shipping over $50

Expected: Conversion rate 5% → 12%
```

**The difference**: Maze finds surface issues, atypica finds root causes.
