### Case 2: SaaS company pricing strategy optimization

**Background**:
- B2B SaaS company providing project management tools
- Current pricing: ¥99/month/user
- Low paid conversion rate (5%), needs optimization

**Problem**:
```

1. Is pricing too high or too low?
2. Which features do users value most?
3. How to design tiers (basic vs professional)?
4. What annual payment discount?
```

**Traditional approach**:
```
Consulting firm quote:
- User interviews: ¥50,000
- Competitive analysis: ¥30,000
- Pricing model design: ¥40,000
- Total: ¥120,000
- Time: 8-10 weeks
```

**Using atypica**:

**Week 1: User insights**
```
Step 1: Scout Zhihu, V2EX discussions
- Discovery: Users dissatisfied with "per-user pricing" (small teams find it expensive)
- Key insight: Want "small team version"

Step 2: Interview 10 target users (high-quality custom AI personas)
- Product managers (3): Willing to pay ¥199/month, but need advanced features
- Entrepreneurs (3): Only willing to pay ¥49/month, features just need to work
- Large company teams (4): Price-insensitive, need stability and security

Core finding:
- Not a "single price" issue, need "tiered pricing"
```

**Week 2: Pricing plan testing**
```
Test 3 pricing plans:

Plan A (current):
- Single price: ¥99/month/user
- Test feedback: Entrepreneurs find it expensive, large companies find features lacking

Plan B (tiered):
- Basic: ¥49/month (limit 5 users, basic features)
- Professional: ¥149/month/user (advanced features)
- Test feedback: Entrepreneurs willing to try basic, but want unlimited users

Plan C (optimized tiers):
- Small team: ¥299/month (unlimited users ≤10, basic features)

- Enterprise: Custom pricing
- Test feedback: ✅ Most popular

Annual discount testing:
- 20% off: Low appeal
- 30% off: Appealing but low profit
- 25% off: ✅ Optimal (high payment willingness, reasonable profit)
```

**Week 3: A/B test preparation**
```
Final pricing plan:
1. Small team: ¥299/month (≤10 users, basic features)
2. Professional: ¥99/month/user (advanced features)
3. Enterprise: Custom pricing (≥50 users)
4. Annual: 25% discount
```

**Business results**:
