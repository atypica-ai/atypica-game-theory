# Fast Insight Agent - Podcast-Driven Rapid Insights

## Core Value

Fast Insight Agent is atypica.AI's podcast-first research agent, focused on **rapidly generating high-quality, in-depth, and engaging podcast content**. Unlike traditional research reports, Fast Insight Agent transforms insights into listenable audio content, enabling users to efficiently consume business and current affairs insights during commutes, workouts, or household chores.

### Three Core Advantages

**1. Podcast-First, Reports Optional**

Traditional research prioritizes written reports as the main deliverable. Fast Insight Agent takes the opposite approach:

- **Primary Output**: Podcast audio + script (15-20 minutes, structured dialogue)
- **Optional Output**: High-density quick-read reports (supplementary detailed data)

**2. Speed Meets Depth**

- **Completion Time**: 5-10 minutes to complete research and podcast generation
- **Podcast Duration**: 15-20 minutes, suitable for commute scenarios
- **Research Depth**: Multi-source information aggregation (web + social media + AI reasoning)

**3. Strict Workflow, Quality Guaranteed**

Fast Insight Agent employs a 5-stage linear process with clear validation checkpoints to ensure content quality:

```
Topic Understanding → Podcast Planning → Deep Research → Podcast Generation → Research Complete
```

- Quickly gathers insights through web search + X search + AI reasoning

- Does not need to understand users' deep motivations and emotions

**Study Agent** - Deep User Research:
- Simulates real humans through AI Personas for deep interviews
- Suitable for research requiring understanding of "why" (e.g., purchase motivation, pain point analysis)

## Workflow: 5-Stage Linear Execution

### Stage 1: Topic Understanding

Quickly collect background information, latest developments, and key concepts about the research topic to lay the foundation for subsequent deep research.

### Stage 2: Podcast Planning

Plan podcast content strategy and deep search strategy, determining:
- Podcast theme and core viewpoints
- Chapter structure and key content
- List of questions requiring deep research
- Information sources and search keywords

### Stage 3: Deep Research

Execute deep research based on podcast planning search strategy to obtain comprehensive insights:
- Web search for latest information and data
- X (Twitter) search for trending social media topics
- AI reasoning to summarize key insights and trends

### Stage 4: Podcast Generation

Generate complete podcast script and audio:
- 15-20 minute structured podcast audio
- High-quality podcast script (independently readable)
- Professional podcast host voice

### Stage 5: Research Complete (Optional Report)

Guide users to access generated podcast content. If users explicitly need more in-depth structured analysis, can generate high-density quick-read reports to supplement data and analysis not detailed in podcast

### How to Make Better Research Requests

**Clarify Podcast Needs**

✅ Good input:
"I want to understand 2025 AI industry trends. Can you generate a 15-minute podcast for me? Focus on LLMs, Agents, and multimodal."

❌ Poor input:
"AI industry trends"

**Provide Background and Context**

✅ Good input:
"I'm a product manager at a SaaS company wanting to understand 2025 enterprise AI application trends. Can you generate a podcast for me? Focus on:
1. Which AI features are most popular with enterprise customers
2. Enterprise customers' willingness to pay
3. What competitors are doing

Podcast needs data support, not too vague."

**Upload Reference Materials**

Fast Insight Agent supports file attachments (images, documents), can generate podcasts based on your uploaded industry reports.

### Podcast and Report Output Formats

**Podcast Script Structure**:
- Opening (1-2 minutes): Hook to attract listeners, topic introduction
- Core Content (10-12 minutes): 3-5 core viewpoints, each including data support, case stories, trend analysis
- Closing (2-3 minutes): Summarize core points, practical insights

**Podcast Audio Quality**:
- 24kHz sample rate, MP3 format
- Professional podcast host voice
- Supports Chinese, English

**Optional Report Format** (high information density):
- Executive Summary: Core findings, key data, practical recommendations
- Detailed Analysis: Data charts, case deep dives, impact analysis
- Data Appendix: Data tables, information sources, methodology notes

**Study Agent**: Report-driven, deep business research (15-30 minutes), conducts deep interviews through AI Personas, suitable for product decisions and user research.

**Selection Advice**: Need podcast → Fast Insight; Need to understand "why" → Study Agent
