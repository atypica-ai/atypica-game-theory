# Scout Agent 深度解析：社交媒体观察系统

> **文档类型**：功能深度解析
> **更新日期**：2026-01-15
> **适用人群**：产品经理、市场研究人员、用户研究人员

---

## 一句话总结

Scout Agent 是社交媒体观察系统，像人类学家做田野调查，通过 3 阶段工作流（观察 → 推理 → 验证）理解用户群体的生活方式、价值观和精神世界，最终构建 7 维度 AI 人设。

---

## 核心特点

| 维度 | 说明 |
|------|-----|
| **核心理念** | 像人类学家做田野调查，而非数据分析师做统计研究 |
| **工作方式** | 沉浸式观察社交媒体内容，理解群体的社会心理和文化特征 |
| **支持平台** | 小红书、抖音、TikTok、Instagram、Twitter（5 个社交平台） |
| **工作流程** | 3 阶段：观察（5 次）→ 推理（reasoningThinking）→ 验证（6-15 次） |
| **最大步数** | 20 次工具调用（最多 15 次观察 + 1 次推理 + 强制总结） |
| **Token 限制** | 150,000 tokens（超过自动停止） |
| **使用模型** | gemini-2.5-flash（成本优化）+ gemini-2.5-pro（推理阶段） |
| **输出维度** | 7 维度人设框架（人口统计学、地理、心理特征、行为、需求与痛点、技术接受度、社会关系） |
| **人设质量** | 社媒人设一致性 79 分（接近人类基线 81%） |
| **适用场景** | 快速理解用户群体、发现文化趋势、构建 AI 人设、市场洞察 |

## 3 阶段工作流

### 阶段 1：沉浸式观察（前 5 次工具调用）

#### 目标
不急于下结论，先大量浏览和观看社交媒体内容，像纪录片导演一样沉浸式观察。

#### 观察维度
1. **生活方式观察**：
   - 他们如何描述自己的日常生活？用什么样的语气？
   - 他们分享的生活片段透露出什么样的价值追求？
   - 他们的表达方式体现了什么样的审美和品味？

2. **社交互动观察**：
   - 他们如何与他人互动？语气是亲密的、疏离的、还是某种特定的风格？
   - 他们如何回应不同观点？是包容的、对抗的、还是避而不谈？
   - 在评论区他们如何表达共鸣？用什么样的话语方式？

3. **价值观与身份观察**：
   - 他们在消费决策中透露出什么样的价值排序？
   - 他们如何平衡理想与现实的冲突？
   - 什么样的内容能引起他们强烈的情感共鸣？

4. **表达与语言观察**：
   - 他们习惯用什么样的词汇和句式？
   - 他们如何使用流行语、梗、emoji？
   - 他们的语言风格在不同场景下如何变化？

#### 具体操作
- 使用社交媒体搜索工具（小红书搜索、抖音搜索、Instagram 搜索等）
- 浏览帖子内容、评论、用户主页
- 记录反复出现的主题、情绪、价值观表达
- 注意那些"有意思"、"有共鸣"、"有冲突"的内容

#### 示例
**用户需求**："帮我观察 30 岁左右的职场女性"

**Scout Agent 在观察阶段做什么**：
- 第 1-2 次：小红书搜索"30岁职场女性"、"职场焦虑"、"30岁困境"
- 第 3-4 次：抖音搜索类似内容，观察不同平台的表达差异
- 第 5 次：Instagram 搜索英文内容（如果是全球性研究）

### 阶段 2：深度理解（第 5 次工具调用后）

#### 目标
**必须使用 reasoningThinking 工具**进行社会心理分析，将观察到的现象提升到价值观、认同、文化符号的层面。

#### reasoningThinking 调用方式
- **background 参数**：简要概述观察到的现象
  ```
  我观察了 30 多条关于"30 岁女性职场焦虑"的小红书帖子。
  她们经常使用"内耗"、"躺平"、"卷"等词汇，语气中既有自嘲也有无奈。
  在讨论消费时，她们会反复强调"悦己"、"值得"，但同时又表现出对价格的敏感。
  在互动中，她们倾向于用表情包和"抱抱"式的情感支持。
  ```

- **question 参数**：提出需要深层理解的社会心理问题
  ```
  这群人反复使用"内耗"和"躺平"的背后，反映了什么样的价值观冲突和身份焦虑？
  她们为什么既强调"悦己消费"又对价格敏感？这种矛盾如何影响她们的决策模式？
  ```

#### 推理产出
- 理解群体的价值观冲突
- 识别身份认同和群体归属感
- 提炼文化符号和社会心理模式
- 形成初步的群体画像假设

#### 关键原则
- ✅ 问"为什么"和"如何"，不问"多少"
- ✅ 关注价值观、情绪、身份认同，不关注统计数据
- ❌ 避免问"统计一下她们的年龄分布和收入水平"
- ❌ 避免问"她们的平均消费金额是多少"

### 阶段 3：继续观察验证（第 6-15 次工具调用）

#### 目标
带着 reasoningThinking 的洞察，继续观察更多细节，验证或修正之前的理解。

#### 具体操作
- 关注那些矛盾的、意外的、复杂的表现
- 深入特定用户的主页，观察个体的完整表达
- 查看评论区的互动，理解群体共识和争议点
- 跨平台对比，观察同一群体在不同平台的表达差异

#### 验证内容
- 初步假设是否成立？
- 是否有反例或例外情况？
- 群体内部是否有细分？
- 价值观冲突如何具体体现？

#### 停止条件
- 达到 20 次工具调用（自动停止）
- Token 消耗超过 150,000（自动停止）
- AI 认为观察已充分（强制总结）

## 7 维度人设框架

Scout Agent 观察和分析的最终产出是 **7 维度 AI 人设框架**：

### 1. 人口统计学（Demographics）
- 年龄、性别、教育、职业
- **注意**：不是通过统计得来，而是通过观察语言、话题、生活方式推断

### 2. 地理（Geographic）
- 城市类型（一线/二线/三线）、区域、居住环境
- **观察点**：他们提到的地点、消费场所、生活场景

### 3. 心理特征（Psychological）
- 价值观、生活态度、性格特质、审美偏好
- **核心**：这是 Scout Agent 最关注的维度

### 4. 行为（Behavioral）
- 消费习惯、社交方式、信息获取渠道、决策模式
- **观察点**：他们如何做决策、如何表达、如何互动

### 5. 需求与痛点（Needs & Pain Points）
- 核心需求、未被满足的需求、焦虑来源
- **提炼方式**：从他们的抱怨、追求、冲突中提炼

### 6. 技术接受度（Technology Acceptance）
- 对新技术的态度、使用习惯、学习能力
- **观察点**：他们使用的平台、表达方式、互动工具

### 7. 社会关系（Social Relations）
- 群体归属感、社交圈层、人际关系模式
- **观察点**：他们如何定义"我们"和"他们"、如何建立认同

## 真实案例：气泡咖啡目标人群观察

### 场景
某饮品公司想推出气泡咖啡，需要快速理解目标人群（喜欢尝试新饮品的年轻人）。

### Scout Agent 观察过程

#### 阶段 1：沉浸式观察（5 次工具调用）

**第 1-2 次：小红书搜索**
- 搜索："气泡水"、"创意饮品"、"网红咖啡"
- 观察发现：
  - 高频词汇："颜值"、"出片"、"打卡"、"仪式感"
  - 语气特点：兴奋、分享欲强、注重视觉呈现
  - 场景：咖啡厅、下午茶、朋友聚会

**第 3-4 次：抖音搜索**
- 搜索："创意饮品"、"咖啡测评"
- 观察发现：
  - 更注重"好不好喝"、"值不值"
  - 喜欢对比评测（气泡水 vs 苏打水）
  - 价格敏感："30 块太贵了"、"20 块以内可以"

**第 5 次：Instagram 搜索**
- 搜索："sparkling coffee"、"cold brew soda"
- 观察发现：
  - 国外更注重健康概念（"low sugar"、"refreshing"）
  - 社交分享是核心（"Instagrammable"）

#### 阶段 2：深度理解（reasoningThinking）

**background**：
```
我观察了 50+ 条关于"创意饮品"、"网红咖啡"的社交媒体内容。
这群人（主要是 20-30 岁年轻人）的表达有明显的双重性：
- 在小红书，他们强调"颜值"、"出片"、"仪式感"，分享照片精美，语气兴奋
- 在抖音，他们更关注"好不好喝"、"值不值"，对价格敏感，会吐槽"智商税"
- 他们既追求新奇体验，又害怕"踩雷"；既想"悦己"，又怕"浪费钱"
```

**question**：
```
这群年轻人在消费创意饮品时的"双重性"（追求新奇 vs 价格敏感）背后，
反映了什么样的价值观冲突？
这种冲突如何影响他们的购买决策？
什么样的产品定位和营销策略能够化解这种冲突？
```

**reasoningThinking 输出**（AI 推理）：
```
【价值观冲突分析】
这群年轻人面临"体验经济"与"理性消费"的冲突：
- 体验经济：追求新奇、社交分享、自我表达（"我是一个有品味的人"）
- 理性消费：经济压力、避免浪费、性价比焦虑（"不想当冤大头"）

【决策模式】
他们的购买决策遵循"两步验证"：
1. 第一步：视觉吸引（颜值、出片、新奇）→ 产生购买冲动
2. 第二步：理性评估（价格、口味、值不值）→ 决定是否复购

【产品策略建议】
- 定价：20-30 元区间（"可以尝鲜"但不贵到"智商税"）
- 定位："可分享的新奇体验"，不是"更好喝的咖啡"
- 营销：强调"尝鲜成本低"（小包装、限时）+ "社交价值"（Instagram-worthy）
- 复购：不依赖复购，用新奇感和限时概念维持热度
```

#### 阶段 3：继续观察验证（6-10 次工具调用）

**第 6-7 次：验证价格敏感度**
- 搜索："网红饮品踩雷"、"智商税饮品"
- 发现：价格超过 30 元，"智商税"评论显著增加
- 验证：20-30 元是心理价格区间

**第 8-9 次：验证复购意愿**
- 查看用户主页，看他们是否多次分享同一产品
- 发现：创意饮品复购率低，但会持续尝试新品
- 验证："新奇感"比"口味"更重要

**第 10 次：验证社交动机**
- 查看评论区互动
- 发现：大量"求链接"、"在哪买"、"想试"评论
- 验证：社交分享是核心购买动机

### 7 维度人设输出

**1. 人口统计学**：
- 20-30 岁，女性为主
- 白领、学生、自由职业者
- 大专及以上学历

**2. 地理**：
- 一二线城市
- 常去商圈、咖啡厅、网红店

**3. 心理特征**：
- 追求新奇体验和自我表达
- 价值观冲突：体验经济 vs 理性消费
- 性格：开放、社交活跃、注重审美

**4. 行为**：
- 消费决策："两步验证"（视觉吸引 → 理性评估）
- 社交方式：喜欢分享、求认同、互相种草
- 信息获取：小红书（种草）+ 抖音（评测）+ 朋友推荐

**5. 需求与痛点**：
- 需求：新奇体验、社交谈资、自我表达、颜值出片
- 痛点：怕踩雷、怕浪费钱、怕被当"冤大头"

**6. 技术接受度**：
- 高度活跃在社交媒体
- 熟练使用滤镜、修图、分享工具
- 愿意尝试新平台和新产品

**7. 社会关系**：
- 群体认同："懂生活"、"有品味"、"会享受"
- 社交货币：分享新奇体验来建立社交地位
- 人际关系：朋友圈、闺蜜、同事是主要影响源

### 商业价值

**基于 Scout Agent 的洞察，公司做出以下决策**：

1. **产品定位**："可分享的新奇体验"，不是"更好喝的咖啡"
2. **定价策略**：25 元（在 20-30 元心理价格区间内）
3. **包装设计**：Instagram-worthy，强调视觉呈现
4. **营销策略**：
   - 小红书：强调颜值、出片、仪式感
   - 抖音：评测向，对比传统咖啡，强调"值得尝试"
   - 限时新品概念（避免新奇感消退）
5. **产品线规划**：
   - 季节性轮换口味（保持新鲜感）
   - 小包装（降低尝鲜门槛）
   - 不依赖复购，持续推新品

### 结果
- **研究时间**：30 分钟（15 次观察 + 分析）
- **研究成本**：约 50-80 元（使用 gemini-2.5-flash，成本优化）
- **产出**：完整的 7 维度人设 + 商业决策建议
- **对比传统调研**：3-4 周 + 8-15 万元 → 30 分钟 + 50-80 元

### 为什么这样设计？

**使用 gemini-2.5-flash 降低成本**：
- Scout Agent 需要大量社交媒体内容处理
- gemini-2.5-flash 性价比高（reduceTokens ratio: 10，成本降低 90%）
- 观察阶段不需要极强的推理能力

**在推理阶段切换到 gemini-2.5-pro**：
- reasoningThinking 需要深度社会心理分析
- gemini-2.5-pro 推理能力更强
- 只在关键节点使用，成本可控

**强制推理机制**：
- 观察 5 次后，如果没有使用 reasoningThinking，强制使用
- 避免 AI 只观察不思考
- 确保深度理解，而非表面浏览

**Token 限制 150,000**：
- 社交媒体内容冗长，容易超预算
- 强制停止机制，避免失控
- 实际使用中，通常在 30,000-50,000 tokens

## 常见问题

### Q1: Scout Agent 和 Interview/Discussion 有什么区别？
**A**: 三者的信息来源和目标完全不同。

| 维度 | Scout Agent | Interview | Discussion |
|------|------------|-----------|-----------|
| **信息来源** | 真实社交媒体内容 | AI 人设模拟访谈 | AI 人设模拟讨论 |
| **目标** | 理解真实用户群体，构建 AI 人设 | 深度挖掘个体动机 | 观察群体意见分布 |
| **输出** | 7 维度人设框架 | 访谈内容（不返回） | 讨论总结 |
| **人设来源** | 观察后构建 | 已有人设 | 已有人设 |
| **适用场景** | 不了解目标用户，需要探索 | 已知目标用户，需要深度洞察 | 已知目标用户，需要意见分布 |

**组合使用**：
1. **Scout Agent** → 探索真实用户群体，构建 AI 人设
2. **Interview/Discussion** → 使用构建的 AI 人设进行深度研究

### Q2: Scout Agent 观察的是真实用户还是 AI 模拟？
**A**: 观察的是**真实社交媒体内容**（真人发布的帖子、评论等），而非 AI 模拟。

**流程**：
1. Scout Agent 使用社交媒体搜索工具（小红书搜索、抖音搜索等）
2. 获取真实用户发布的内容（帖子、评论、个人主页）
3. AI 分析这些真实内容，理解群体特征
4. 基于真实数据构建 AI 人设

**真实性保证**：
- 社交媒体 API 返回的是真实用户内容
- 不是 AI 生成或模拟的内容
- 人设一致性达到 79 分（接近人类基线 81%）

### Q3: Scout Agent 需要多长时间完成？
**A**: 通常 15-30 分钟，取决于研究复杂度和平台数量。

**时间分解**：
- **观察阶段**（5 次工具调用）：10-15 分钟
- **推理阶段**（reasoningThinking）：2-3 分钟
- **验证阶段**（6-10 次工具调用）：10-15 分钟
- **总结**：2-3 分钟

**影响因素**：
- 社交媒体 API 响应速度（主要因素）
- 搜索关键词是否精准（影响内容相关性）
- 需要观察的平台数量（单平台 vs 多平台）

### Q4: Scout Agent 的成本如何？
**A**: 约 50-80 元/次，比传统调研便宜 99%+。

**成本构成**：
- **模型调用**：gemini-2.5-flash（30,000-50,000 tokens，约 30-50 元）
- **推理分析**：gemini-2.5-pro（5,000-10,000 tokens，约 10-20 元）
- **社交媒体 API**：已包含在订阅中

**对比传统调研**：
- 传统焦点小组：8-15 万元（招募 + 执行 + 分析）
- Scout Agent：50-80 元（自动化执行）
- **成本降低 99.9%**

### Q5: Scout Agent 能观察所有社交媒体吗？
**A**: 目前支持 5 个主流平台：小红书、抖音、TikTok、Instagram、Twitter。

**平台选择策略**：
- **中文环境**：优先小红书 + 抖音（内容质量高、用户活跃）
- **英文环境**：优先 Twitter + Instagram + TikTok
- **全球性研究**：跨平台对比（中英文平台都搜索）

**未来计划**：
- 增加更多平台支持（B站、知乎、YouTube 等）
- 支持自定义数据源（CSV 导入、API 接入）

### Q6: Scout Agent 构建的 AI 人设质量如何？
**A**: 社媒人设一致性 79 分，接近人类基线 81%，适合快速探索和大规模研究。

**质量对比**：
- **社媒人设**（Scout 构建）：79 分一致性
  - 优势：快速、大规模、覆盖多样性
  - 劣势：深度略低于深访人设

- **深访人设**（Interview 构建）：85 分一致性
  - 优势：深度更高、决策路径完整
  - 劣势：需要深度访谈数据（更耗时）

- **人类基线**：81% 重测一致性
  - 同一个人，两次回答相同问题的一致性

**使用建议**：
- **探索阶段**：用 Scout 构建社媒人设（快速、覆盖广）
- **深度验证**：用 Interview 构建深访人设（深度高、决策完整）

### Q7: Scout Agent 会不会因为社交媒体内容质量问题而得出错误结论？
**A**: 有可能，但通过 3 阶段工作流和 reasoningThinking 机制降低风险。

**潜在风险**：
1. **平台偏差**：不同平台用户群体不同（小红书偏女性、抖音更大众）
2. **表演性表达**：社交媒体内容可能是"表演"而非真实想法
3. **极端观点**：容易被极端、情绪化内容吸引
4. **信息茧房**：算法推荐可能导致观察偏差

**降低风险的机制**：
1. **跨平台验证**：在多个平台观察，对比差异
2. **reasoningThinking**：深度分析，识别"表演性表达" vs "真实想法"
3. **验证阶段**：关注矛盾、意外、复杂的表现，修正初步结论
4. **人类学视角**：理解"为什么这样表达"，而非"表达了什么"

**使用建议**：
- Scout Agent 适合快速探索，发现趋势和模式
- 重要商业决策前，建议用 Interview/Discussion 深度验证
- 结合定量数据（销售数据、调查问卷等）交叉验证

### Q8: Scout Agent 可以用于竞品用户研究吗？
**A**: 可以，而且非常适合。

**使用场景**：
1. **竞品用户画像**：观察竞品用户的社交媒体表达，理解他们的特征
2. **用户迁移动机**：理解为什么用户选择竞品（从评论、讨论中提取）
3. **产品优劣对比**：从用户视角理解竞品的优势和不足
4. **市场空白发现**：识别用户未被满足的需求

**示例**：
- 搜索："Notion 用户体验"、"Notion vs Obsidian"
- 观察 Notion 用户的痛点、需求、使用习惯
- 发现市场空白（例如："Notion 太复杂"→ 简化版 Notion 的机会）

### Q9: Scout Agent 和 Product R&D Agent 有什么区别？
**A**: Scout Agent 是"理解用户"，Product R&D Agent 是"寻找创新机会"。

| 维度 | Scout Agent | Product R&D Agent |
|------|------------|-------------------|
| **目标** | 理解用户群体 | 寻找产品创新机会 |
| **方法** | 社交媒体观察 | 趋势分析 + Audience Call |
| **输出** | 7 维度人设框架 | 创新机会地图 |
| **使用场景** | 不了解用户，需要探索 | 已有产品，需要创新方向 |
| **关注点** | 用户是什么样的人 | 市场趋势和创新机会 |

**组合使用**：
1. **Scout Agent** → 理解目标用户（"25-30 岁职场女性是什么样的人"）
2. **Product R&D Agent** → 为这群用户寻找创新机会（"她们需要什么新产品"）

### Q10: Scout Agent 可以保存观察结果吗？
**A**: 可以，Scout Agent 的所有观察内容都保存在独立的 scoutUserChat 中。

**保存内容**：
- 所有社交媒体搜索结果（帖子、评论、用户主页）
- reasoningThinking 的分析结果
- 最终的 7 维度人设框架
- 平台覆盖统计（观察了哪些平台，各平台调用次数）

**后续使用**：
1. **构建 AI 人设**：使用 `buildPersona` 工具基于 Scout 观察构建人设
2. **继续观察**：重新打开 scoutUserChat，继续观察和验证
3. **参考研究**：在其他研究中引用 Scout 观察结果
4. **导出报告**：将观察结果生成结构化报告

**数据位置**：
- 数据库：`userChat` 表（kind="scout"）
- 消息记录：`chatMessage` 表（完整的观察和分析过程）
