# Plan Mode：AI 研究规划助手

## 产品价值

Plan Mode 将模糊的调研想法转化为可执行的研究方案，只需一次对话。

**核心亮点**：
- **自然对话代替填表**：像和顾问聊天一样描述需求
- **AI 自动选择方法**：无需学习 JTBD、KANO 等专业术语
- **一键确认开始**：看完计划直接执行，无需反复选择
- **5分钟完成设计**：传统需要 3-5 天的研究设计工作

**适用场景**：产品经理做用户调研、市场人员做竞品分析、创业者验证产品想法

## 一、核心对比：Plan Mode vs 传统调研流程

| 维度 | 传统调研公司流程 | 传统工具（问卷星/SurveyMonkey）| atypica.AI Plan Mode |
|------|----------------|------------------------------|---------------------|
| **开始方式** | 客户填写需求表（Word/Excel） | 用户选择问卷模板 | 自然对话："我想了解XX" |
| **需求澄清** | 项目经理电话沟通（1-2 轮） | 无澄清，用户自行决定 | AI 逐步澄清（5-10 轮对话） |
| **方法选择** | 专家提供 2-3 个方案选项 | 用户自己查资料学习 | **AI 自动判断最优方案** |
| **时间成本** | **3-5 天**（含沟通、提案、修改） | 1-2 天（学习+设计） | **5-10 分钟** |
| **专业门槛** | 需要理解调研术语 | 需要学习工具和方法论 | **零门槛**（对话即可） |
| **确认次数** | 多次确认（需求→方案→细节） | 无确认（直接发布） | **一次确认**（完整计划） |

**关键差异**：
- **传统流程**："填表 → 等待 → 选方案 → 再等待 → 开始"（3-5 天）
- **Plan Mode**："对话 → AI 判断 → 确认 → 开始"（5-10 分钟）

## 二、工作流程：从想法到研究方案

### 四个阶段

**阶段 1：对话澄清（5-10 轮）**
AI 逐步提问，了解研究对象、场景、关注点。不是一次性填表，而是自然对话。

**阶段 2：背景调研（2-5 分钟，可选）**
当 AI 判断需要了解行业背景时，自动搜索最新市场趋势和竞品动态。

**阶段 3：智能判断（秒级）**
AI 自动判断研究类型、选择分析框架、决定研究方式、配置人设数量。

**阶段 4：展示计划（一键确认）**
展示完整研究方案，用户确认后自动开始执行。

### 对话澄清的体验差异

**特点**：**逐个澄清，不是批量提问**

#### 传统方式（❌ 不好的体验）

```
系统: 请选择：
1. 研究对象？ (下拉菜单：18-25岁 / 26-35岁 / ...)
2. 研究场景？ (下拉菜单：购买决策 / 使用体验 / ...)
3. 关注维度？ (多选框：价格 / 品牌 / 口碑 / ...)
4. 分析框架？ (单选：JTBD / KANO / STP / ...)
5. 研究方式？ (单选：访谈 / 讨论 / 观察 / ...)
```

**问题**：
- ❌ 用户不知道这些术语是什么意思
- ❌ 不知道该怎么选择
- ❌ 填完发现选错了，要重新填

#### Plan Mode 方式（✅ 好的体验）

```
用户: 我想了解年轻人对气泡咖啡的看法

AI: 好的！你说的年轻人，大概是哪个年龄段？
    A. 18-22岁（大学生）
    B. 23-28岁（职场新人）
    C. 都想了解

用户: B，23-28岁职场新人

AI: 明白了。你主要想了解他们在什么场景下的选择？
    是日常购买的时候，还是第一次尝试的时候？

用户: 日常购买的场景

AI: 好的。你更关注他们的哪些方面？
    A. 为什么选择气泡咖啡（动机）
    B. 对价格的接受度
    C. 对品牌的偏好
    D. 以上都关注

用户: 主要是 A 和 C，想知道为什么选它，以及品牌偏好

AI: 了解了！让我快速搜索一下气泡咖啡市场的背景...
    [进行 webSearch...]

    根据你的需求，我推荐使用 **JTBD (Jobs To Be Done) 框架** 来理解他们"雇佣"气泡咖啡解决什么问题，
    并通过 **一对一深度访谈** 来挖掘真实动机。

    [展示完整计划]

    是否开始执行？
```

**为什么这样更好**：
- 逐步澄清，用户只需回答问题，无需理解专业术语
- AI 自动判断最优方法，准确率超过 95%
- 随时可以修改，对话即调整

### 智能判断示例

当需要了解行业背景时，AI 会自动搜索最新信息：

```
用户: 我想了解 Z 世代对元宇宙的态度

AI: 好的，让我先快速了解一下元宇宙的最新动态...

    根据最新信息：
    - 元宇宙市场 2024 年增长 35%
    - 主要应用：游戏、社交、虚拟演唱会
    - Z 世代占用户的 62%

    现在我对背景有了了解。你说的 Z 世代，具体是指...
```

AI 会自动判断研究类型和方法：

- **快速洞察**："快速了解年轻人对 AI 的态度，做成播客" → 生成播客式内容
- **产品创新**："看看小红书上咖啡的新玩法" → 社交媒体观察
- **深度理解**："了解高端用户为什么流失" → 深度访谈，使用 JTBD 框架
- **概念测试**："比较气泡咖啡和冷萃咖啡" → 群体讨论，使用 KANO 框架

## 五、使用技巧

### 对话技巧

#### ✅ 好的沟通方式

```
用户: 我想了解年轻人对气泡咖啡的看法

AI: 你说的年轻人，大概是哪个年龄段？

用户: 23-28岁

AI: 主要想了解他们在什么场景下的选择？

用户: 日常购买
```

**推荐做法**：
- 逐步澄清，每次回答一个问题
- 简洁明确，让 AI 引导对话节奏

#### ❌ 避免的方式

```
用户: 就是年轻人啊，18到35岁都算，想知道他们对各种产品的看法，
      包括咖啡、茶饮、果汁，还有消费习惯...
```

**问题**：目标过于宽泛，一次提供太多信息

### 修改计划的方式

#### 场景 1：调整研究对象

```
[AI 展示完整计划]

用户: 等等，我想把年龄扩大到 18-28 岁

AI: 好的，那我重新调整一下人设配置...
    [更新计划，再次展示]

    现在研究对象调整为 18-28 岁，人设数量增加到 10 个，
    覆盖大学生和职场新人两个群体。

    是否开始执行？
```

直接说明需要调整的地方，AI 会立即响应。

### 常见问题

**如何避免调整次数过多？**
- 明确说出问题或目标，不要让 AI 猜测
- 让 AI 引导对话节奏，按需提供信息
- 不要一次性提供过多背景信息

**如果计划不满意怎么办？**
- 先说明具体哪里需要调整，而非直接取消
- AI 会根据您的反馈重新生成计划

## 六、常见问题

### Plan Mode 的判断准确率如何？

准确率超过 95%。如果判断错误，您可以在对话中说明，AI 会重新调整。

### 支持哪些语言？

目前支持中文和英文，AI 会自动检测对话语言。

### 可以跳过 Plan Mode 直接开始吗？

不可以。Plan Mode 是所有研究的必经入口，确保意图明确后再执行。

### 能处理多目标研究吗？

可以，但 AI 会建议拆分成多个阶段，确保每个目标都能充分实现。

- **节省时间**：3-5 天压缩到 5-10 分钟
- **零门槛**：无需学习专业方法论
- **完全透明**：理解所有判断逻辑

- 产品经理做用户研究
