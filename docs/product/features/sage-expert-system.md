# Sage：可进化的领域专家

**一句话总结**：上传文档/网页，AI 自动构建领域专家，可持续对话和进化。

## 完整生命周期

1. **Create**：上传知识源（PDF/网页/文本）
2. **Extract**：AI 自动提取知识，理解整体
3. **Chat**：基于知识库对话，提供专业建议
4. **Discover Gaps**：AI 分析对话，发现知识盲区
5. **Evolve**：针对盲区提问你，更新知识库

## vs. 普通文档问答

- 普通问答：检索相关片段 → 回答
- Sage：理解整体知识 → 发现不足 → 主动进化

## 适用场景

- 构建企业内部专家（如"我们公司的品牌指南专家"）
- 将顾问/导师的知识固化（上传他们的文章/书籍）
- 持续学习型专家（如"巧克力行业专家"不断吸收新知识）

## 真实案例

**创建 UX 设计专家**：
1. **上传**：3 本 UX 设计书籍 + 10 篇设计博客
2. **提取**：AI 自动总结设计原则和方法论
3. **使用**：咨询"如何设计一个好的注册表单？"
4. **发现盲区**：AI 回答后发现缺少"移动端表单"知识
5. **进化**：AI 提问"移动端表单有什么特殊考虑？"，你回答后更新知识库
6. **下次**：能够回答移动端相关问题

**价值**：这不是静态的知识库，而是会持续进化的领域专家。
