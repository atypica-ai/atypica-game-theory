# 参考研究 + 文件附件 - 让 AI 记住你的每次研究

## 为什么需要这两个功能?

做研究时,你是否遇到过这些问题:

- **重复劳动**:每次新研究都要重新调研基础信息,浪费时间
- **上下文断裂**:AI 不记得上次的发现,无法深入分析
- **资料利用困难**:手里有报告和数据,却只能复制粘贴,费时费力

atypica.AI 通过**参考研究**和**文件附件**两大功能解决这些问题:

1. **参考研究**:AI 记住你之前的所有研究,可以直接基于旧发现继续深入
2. **文件附件**:上传 PDF 报告、Excel 数据、产品截图,AI 自动理解并整合到分析中

这两个功能让研究**可积累、可复用、可深入**,而不是每次从零开始。

## 功能 1:参考研究

### 这个功能能做什么?

**简单来说**:AI 会记住你每次研究的过程和发现,在新研究中可以直接引用,避免重复劳动。

**类比**:
- **传统方式**:每次写论文都要从第一章开始写,重复介绍背景
- **参考研究**:直接引用之前的章节,"基于上次研究,我们发现...",快速进入核心论证

### 工作原理

每次研究完成后,atypica.AI 会自动生成一份"研究日志",记录:
- 研究主题是什么
- 调研了哪些内容
- 得出了什么结论

在新研究中,你可以选择引用之前的研究,AI 会自动加载那次研究的所有背景知识,然后基于这些已知信息继续深入。

### 适用场景

#### ✅ 最适合的场景

**1. 渐进式深入研究**
```
研究 A:宏观市场分析 → 研究 B:用户痛点(引用 A) → 研究 C:产品方案(引用 A+B)
```

**2. 跨时间对比**
```
研究 A:2024 Q4 趋势 → 研究 B:2025 Q1 趋势对比(引用 A,自动对比变化)
```

**3. 多维度综合**
```
研究 A:产品功能分析
研究 B:用户痛点分析
研究 C:改进方案(引用 A+B,结合功能和痛点给出方案)
```

**4. 团队协作**
```
成员 A:市场调研 → 成员 B:竞品分析(引用 A) → 成员 C:产品定位(引用 A+B)
```

#### ❌ 不适合的场景

- **无关联的新研究**:研究新能源车 vs 研究咖啡行业(两者无关)
- **过时的研究**:2023 年的数据对 2025 年研究价值有限
- **矛盾的结论**:两个研究结论相反,引用会让 AI 混淆

### 最佳实践

**推荐使用方式**:

**场景:产品创新研究**
```
第 1 步:宏观市场研究
"研究 2025 年 AI 助手市场趋势"
→ 生成研究 A(市场规模、玩家、趋势)

第 2 步:竞品分析(引用研究 A)
"基于刚才的研究,分析 ChatGPT、Claude、Gemini 的差异"
[参考研究:研究 A]
→ AI 已知市场背景,直接进入竞品分析

第 3 步:产品定位(引用研究 A + B)
"基于市场和竞品分析,给出产品定位方案"
[参考研究:研究 A, 研究 B]
→ AI 结合市场和竞品,给出定位方案
```

**优势**:
- 每次研究都站在之前的肩膀上
- 避免重复调研基础信息
- 研究深度逐步递增

### 能力边界

**能做什么**:
- 自动记住你的所有研究
- 避免重复调研已知信息
- 同时引用多个研究(最多 5 个)
- 团队成员可以引用彼此的研究

**不能做什么**:
- 不能引用别人的研究(隐私保护)
- 引用的内容是静态的(不会实时更新)
- 不能只引用研究的一部分(只能全部引用)

## 功能 2:文件附件

### 这个功能能做什么?

**简单来说**:上传 PDF 报告、Excel 数据、产品截图,AI 会自动提取和理解内容,并整合到研究分析中。

**类比**:
- **传统方式**:手动复制粘贴文档内容,100 页报告只能选几段
- **文件附件**:直接上传 PDF,AI 自动提取全部关键内容

### 支持的文件类型

#### 1. 图片文件(最多 5 张)

**支持格式**:JPEG, PNG, GIF, WebP, BMP, SVG

**适用场景**:
- **产品截图**:App 界面、网页设计、功能对比
- **数据图表**:市场趋势图、用户调研结果图
- **竞品分析**:竞品产品截图、设计对比
- **用户反馈**:社交媒体评论截图、用户反馈图

#### 2. 文档文件(最多 3 个)

**支持格式**:PDF, TXT, CSV

**适用场景**:
- **行业报告**:麦肯锡、艾瑞、Gartner 等咨询公司报告
- **内部文档**:公司战略、产品需求文档、会议纪要
- **数据文件**:用户调研数据(CSV 表格)
- **研究论文**:学术论文、技术白皮书

### 文件限制

- **单个文件**:最大 3MB
- **图片**:最多 5 张
- **文档**:最多 3 个
- **总大小**:最大 50MB

### 文档处理流程

上传文档后,AI 会自动:
1. **提取文本**:从 PDF 中提取所有文字内容
2. **智能压缩**:保留所有关键信息(数据、观点、结论),删除冗余内容
3. **整合分析**:将文档内容融入研究,结合其他信息给出分析

**处理时间**:
- 小文档(< 20 页):1-2 分钟
- 中等文档(20-50 页):2-3 分钟
- 大文档(50-100 页):3-5 分钟

- 扫描版 PDF(纯图片)识别率低
- 超大文档(> 200 页)可能丢失部分信息

- ✅ 正式行业报告
- ✅ 结构化文档(有章节、标题)

- ❌ 扫描版 PDF(纯图片)
- ❌ 加密 PDF
- ❌ 超大文档(> 200 页)

## FAQ

### 1. 参考研究和文件附件有什么区别?

- **参考研究**:你之前在 atypica.AI 完成的研究
- **文件附件**:你上传的外部资料(报告、截图等)

**类比**:
- 参考研究 = 你之前写的论文草稿
- 文件附件 = 你收集的参考资料

### 2. 可以同时使用两个功能吗?

**可以!**这是最强大的组合:

```
研究 A:市场调研
[附件:行业报告 1.pdf, 行业报告 2.pdf]

研究 B:竞品分析
[参考研究:研究 A] + [附件:竞品截图.png, 竞品报告.pdf]
→ AI 已知市场背景 + 新分析竞品
```

### 3. 文档处理需要多长时间?

- 小文档(< 20 页):1-2 分钟
- 中等文档(20-50 页):2-3 分钟
- 大文档(50-100 页):3-5 分钟

处理在后台进行,你可以继续输入研究问题。

### 4. 为什么要压缩文档?

**原因**:
- AI 上下文窗口有限
- 提高理解效率
- 降低成本

**效果**:
- 100 页 PDF → 压缩 82%
- 保留 100% 关键信息(数据、观点、结论)

### 5. 可以上传 Word/Excel/PPT 吗?

**目前不支持**,替代方案:
- Word → 导出为 PDF
- Excel → 导出为 CSV
- PPT → 导出为 PDF

### 6. 参考研究有数量限制吗?

**最多 5 个**

**最佳实践**:
- 引用 1-2 个(最常见)
- 引用 3-4 个(多维度分析)
- 避免 5 个以上(上下文过载)
