# Reference & Attachments：研究延续

**一句话总结**：关联历史研究 + 上传文档，AI 基于已有洞察继续深挖。

## Reference Study（参考研究）

- 关联之前的研究项目 → AI 自动读取研究日志
- 避免重复工作，基于已有发现继续探索

## File Attachments（文件附件）

- 支持类型：PDF/TXT/CSV/图片（最多 10MB/文件）
- AI 自动提取文档内容
- 支持上传竞品报告、原型图、用户反馈等

## 适用场景

- **连续研究**：先做市场分析，再做产品验证，两者关联
- **文档分析**：上传竞品报告，AI 提取关键信息
- **原型测试**：上传产品原型图，讨论用户反应

## 真实案例

**连续研究**：

**第 1 次研究**："气泡咖啡市场机会分析"
- 输出：市场有机会，目标用户是 25-35 岁都市女性

**第 2 次研究**："基于市场机会，设计产品包装"
- 关联第 1 次研究 + 上传 3 个包装设计稿
- AI 基于已知的目标用户特征，讨论哪个包装更吸引她们
- 输出：包装建议，无需重新研究目标用户

**价值**：研究可以累积，不是每次从零开始。
