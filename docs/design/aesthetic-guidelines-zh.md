# 越不AI越AI：atypica的设计哲学

2025年12月28日


## 我们为什么这样做设计

atypica.AI用AI模拟消费者，把市场研究从weeks压缩到minutes。但这个描述容易误导人——好像我们在炫耀技术，好像速度本身就是价值。其实不是的。

我们真正做的是帮助人们理解人。消费者如何思考、如何感受、在真实生活场景中如何做决策。AI是工具，产出的是对人性的洞察。这个本质决定了我们的视觉语言应该是什么样的。

所以设计的核心问题变成：如何用视觉呈现对人的深刻理解？

现在市场上所有AI产品都在用霓虹色、3D渲染、科技感渐变来"看起来很AI"。这是一种本能反应，也是一种偷懒——既然做AI产品，那就应该看起来futuristic，对吧？我们选择了相反的方向。


### 越不AI越AI

这不是玩文字游戏，是我们设计哲学的核心。

因为我们模拟的是人，研究的是人，服务的是理解人这件事本身。所以最准确的视觉语言应该回归人本身——但不是用柔和、克制、安全的方式。应该有力量、有戏剧性、有视觉野心。应该能抓住注意力。

当别人都在用廉价的科技感炫技时，我们用专业的视觉语言——电影摄影、建筑摄影、编辑设计。这个"反AI视觉语言"，恰恰是最诚实的AI产品表达。

关键洞察：避开AI陈词滥调不意味着变得安静克制。而是用成熟的手法创造冲击力。


## 真实大于合成，但要有戏剧性

我们研究真实的消费者，不是抽象的数据点。这意味着摄影应该优于3D渲染，自然材质应该优于人工合成，人的存在应该优于抽象符号。

但"真实"不是只能纪实风格。一张戏剧性打光的电影化肖像依然是真实的——人是真的，光是真的，这个时刻即使是摆拍也有真实性。我们避免的是CGI渲染的合成感、信息图表的塑料质感、模板设计的廉价拼贴。

想想那些最好的电影摄影——高度风格化、戏剧性打光，但本质上关于人的情感和存在。这是真实。一个漂浮的3D全息图配粒子特效？那是合成垃圾。


## 力量和深度

从weeks压缩到minutes，这个价值不在于快，在于能更深入地理解人。所以视觉也应该有深度和力量。

力量意味着视觉冲击力。戏剧性的色彩对比。强烈的几何构图。能停止滚动的电影化打光。建筑般的尺度和纪念性。我们在社交媒体上竞争，注意力以零点几秒计算。微妙和柔和不够用。

深度意味着经得起细看的内容。能感受到的质感。有张力和呼吸感的构图。创造情绪和意义的色彩关系。放大了依然站得住的细节。

但力量和深度加在一起不等于复杂。有时最有力的图像也是最简单的——巨大建筑空间里的单个人影，戏剧性对比的两种颜色，一束光线切开阴影。


## 专业但不疏远

我们的客户是决策者，需要咨询级的专业度。但我们的研究对象是真实的人，需要人类学的共情。这两者不矛盾——它们要求视觉语言既有权威感又有人性。

想象顶级咨询公司的严谨被世界级新闻摄影师拍摄。学术精确度用电影场景打光。咨询公司的权威性配编辑杂志的美学。这个组合很难但必要——这正是atypica.AI应该有的调性。

专业不是死板的企业模板。人性不是软绵绵的温情。我们可以同时有力量和共情，同时戏剧化和诚实。


## 色彩作为戏剧，不是装饰

这是很多设计指南搞错的地方：他们说"柔和色调"，人们听成了"避免色彩"。不是这样的。

我们避免的是无意义的彩色点缀——往信息图表上加蓝色和橙色"让它跳出来"。随机的彩虹渐变。没有目的的霓虹高光。色彩作为装饰。

我们拥抱的是色彩作为戏剧。冷暖对比创造视觉张力和分离感。有目的的大胆饱和度——无论是在社交媒体上停止滚动，还是创造情感氛围。

想想电影里的色彩运用：科幻电影的冷暖对比，当代爱情片的暖色调，艺术电影的精确配色。这些用色大胆但成熟。饱和度是有意的、受控的、有意义的。

对于社交媒体封面，我们需要能吸引注意的色彩。可以用对比色创造视觉冲击。但要通过克制的手法实现——不是通过渐变和图形叠加。

对于报告，我们可以更克制——黑、白、灰，也许一个单一强调色。但即使在那里，当色彩出现时，它应该有目的和力量。


## 视觉语言光谱

不同内容需要不同方法，但所有选择回到一个问题：这是否诚实而有力地传达了本质？

**电影摄影**：用于消费者洞察、品牌故事、社交媒体冲击。戏剧性打光，强烈色彩对比，环境中的人的存在。参考世界顶级电影摄影师和当代视觉艺术家的作品。这是我们处理情感和以人为中心内容的主要语言。

**建筑摄影**：用于战略分析、结构化思考、数据驱动洞察。几何构图，光影戏剧，纪念性尺度配人的存在作为锚点。参考国际建筑摄影大师的作品。当我们需要传达系统性思考和专业权威时。

**纪实摄影**：用于真实消费者时刻、田野研究、人的故事。自然光，真实环境，诚实时刻。参考世界顶级新闻摄影机构和人类学田野摄影传统。当真实性和现实主义最重要时。

**编辑设计**：用于报告、演示、专业文档。成熟排版，克制色彩，精心构图。参考现代主义设计经典、欧洲极简主义传统。当我们需要咨询级精确度时。

**线条插画**：用于封面图、社交媒体图、简洁的品牌传播。清晰的线条，克制的色彩（2-3种主色），现代感，充分留白。适合需要快速传达信息、在小尺寸下保持清晰的场景。简洁优于复杂，避免过度装饰。

关键是为每个工作选择合适的工具，同时在所有触点保持视觉连贯性。


## 具体执行指南

### 构图原则

**三分法则是起点，不是终点**：主体元素可以放在画面的三分之一处，但要有理由打破这个规则。偏离中心创造张力，但要确保画面平衡。

**留白分配**：
- 社交媒体封面：至少留出10-15%的安全边距
- 报告内页：页边距至少占页面的12-15%
- 单个元素周围：保持至少相当于元素自身宽度30%的呼吸空间

**人物位置和比例**（当画面中有人时）：
- 建筑/环境场景：人物占画面高度的1/4到1/3，不要小于1/5
- 中景人物：人物占画面高度的1/2到2/3
- 避免：人物太小成为点缀（除非特定主题需要表现渺小感）

**视线引导**：
- 人物视线方向应指向画面的留白区域，创造叙事空间
- 几何线条应汇聚或引导视线到关键区域
- 光线方向应强化构图的动势

### 色彩使用细则

**主色方案**：
- 单色+强调：大面积中性色作为基础，单一有目的的色彩作为强调
- 对比配色：暖色调和冷色调在空间上分离使用（如前景暖、背景冷）
- 节制配色：选择有限的颜色（2-3种主要色彩），每种都有明确目的

**色彩选择原则**：
- 每个颜色必须有存在的理由，不是"让画面丰富"
- 高饱和度可以用，但要服务于吸引注意或传达情绪
- 冷暖对比比多色堆砌更有力量
- 环境色和光源色应该来自真实场景（天空、日落、街灯、室内光）

**禁止的色彩使用方式**：
- 霓虹色的无目的堆砌（除非主题明确需要）
- 彩虹渐变（超过3种高饱和色的渐变）
- 为了"平衡"而加入的无意义色彩
- 缺乏对比度的模糊配色（看不出色彩关系）

### 光影处理

**光源设定**：
- 单一主光源：创造明确的明暗关系，阴影占画面30-50%
- 环境光+主光源：主光源创造方向性，环境光填充阴影但不消除对比
- 避免：无方向感的均匀照明

**光线角度**：
- 侧光（45-60度）：最能展现质感和立体感
- 顶光（70-90度）：创造戏剧性和几何感
- 逆光：用于剪影和轮廓，适合概念性内容

**明暗对比**：
- 高对比：最亮和最暗区域的明度差>60%，用于戏剧性场景
- 中对比：明度差40-60%，用于大部分内容
- 低对比：明度差<40%，仅用于特定情绪表达

### 插画执行指南

**线条质量**：
- 清晰、干净的线条，避免模糊或毛糙
- 线条粗细要有层次，但不要过于复杂
- 几何图形要精确，但保持手绘感

**色彩原则**：
- 优先2-3种主色，每个颜色都要有明确目的
- 可以用对比色创造视觉张力（不要死板遵循特定配色）
- 避免：随机彩虹渐变、过多颜色堆砌、无意义的装饰色

**简洁与留白**：
- 简洁优于复杂，不要填满整个画面
- 留白是设计的一部分，创造呼吸感
- 避免过度装饰、信息过载

**元素融入**：
- 报告封面：可融入文档、笔记、图表轮廓等符号（占15-25%，不抢眼）
- 播客封面：可融入耳机、麦克风、声波等符号（占15-25%，不抢眼）
- 元素应作为点缀，不是主角

**小尺寸优化**：
- 核心元素要足够大，在100x100px时仍清晰可辨
- 避免细小元素拥挤
- 测试缩略图效果

### 图像技术规格

**输出尺寸**：
- 社交媒体封面（16:9）：1920x1080px 最低，2560x1440px 推荐
- 方形（1:1）：1080x1080px 最低，2000x2000px 推荐
- 报告插图：300 DPI，最小边长2000px
- 网站hero图：3840x2160px（4K）用于关键位置

**文件格式**：
- 摄影作品：JPEG（质量90-95%）或WebP
- 有透明需求：PNG
- 印刷用途：TIFF或高质量PDF

**压缩标准**：
- 社交媒体：文件大小<500KB，但不牺牲可见质量
- 网站使用：<1MB，关键图片可以到2MB
- 印刷：不压缩，保持原始质量

### 决策流程图

**第一步：确定内容类型**
```
消费者洞察/品牌故事 → 电影摄影语言
战略分析/数据报告 → 建筑摄影语言
真实故事/案例研究 → 纪实摄影语言
专业文档/演示 → 编辑设计语言
封面图/社交媒体图 → 线条插画语言
```

**第二步：选择色彩策略**
```
需要强冲击力（社交媒体）→ 高饱和度对比配色
需要专业权威（报告）→ 黑白灰+单一强调色
需要情感温度 → 暖色调为主
需要理性分析 → 冷色调为主
```

**第三步：确定人物处理**
```
主题关于人的行为/情感 → 人物占1/2-2/3画面
主题关于环境/系统 → 人物占1/4-1/3画面
主题关于抽象概念 → 可以无人物或剪影
```

**第四步：检查清单**
- [ ] 主体元素是否清晰？（5米外能看清关键信息）
- [ ] 色彩是否有目的？（能说出每个颜色为什么在这）
- [ ] 构图是否有张力？（不是居中对称的死板）
- [ ] 留白是否充足？（画面不拥挤）
- [ ] 光影是否有方向性？（不是均匀平光）
- [ ] 如果有人：人是否有存在感？（不是装饰点缀）
- [ ] 避开禁止项了吗？（无霓虹渐变、3D堆砌、信息图表式）
- [ ] 小尺寸下是否清晰？（缩略图测试）

### 常见场景应用

**场景1：产品发布社交媒体图**
- 视觉语言：电影摄影
- 构图：产品/场景占画面60-70%，留白30-40%
- 色彩：产品环境的自然色+单一强调色背景
- 光影：强侧光或逆光，创造轮廓和质感
- 人物：如有，占画面1/3，与产品互动

**场景2：市场研究报告封面**
- 视觉语言：建筑摄影或纪实摄影
- 构图：主体在下1/3或左1/3，大量天空/留白
- 色彩：冷暖对比，饱和度克制
- 光影：自然光，有明确方向
- 人物：如有，占画面1/4-1/3，展现沉思或观察状态

**场景3：数据分析图表页面**
- 视觉语言：编辑设计
- 构图：图表占页面70%，留白30%用于标注
- 色彩：黑白灰+单一数据强调色
- 避免：多彩柱状图、饼图的彩虹配色
- 重点：通过尺寸和位置建立层级，不是颜色

**场景4：播客/视频节目封面**
- 视觉语言：电影摄影
- 构图：人物（如有）占画面50-60%，偏向一侧
- 色彩：大胆对比，暖色人物+冷色背景或反之
- 光影：戏剧性打光，明暗对比强烈
- 氛围：电影海报感，但不脱离真实

### 快速自检标准

拿着你的设计，问自己：

**1. 能在0.5秒内抓住注意力吗？**
测试方法：快速滚动，它是否让你停下来？

**2. 有专业感吗？**
测试方法：和顶级品牌/出版物的视觉并排，是否不丢人？

**3. 避开AI陈词滥调了吗？**
测试方法：把它放在一堆AI产品图里，是否明显不同？

**4. 主题准确吗？**
测试方法：不看文字，视觉本身是否传达了正确的情绪/主题？

**5. 在各种尺寸下都行吗？**
测试方法：缩小到100x100px，核心元素是否依然清晰？


## 人作为主体，不是比例参照

当我们说"越不AI越AI"时，我们的意思是视觉应该聚焦于理解人。所以人应该是主体，不是事后想起来的。

这不是说要填满画面的脸部特写。而是说当人出现在图像中时，他们应该有存在感和目的。他们让构图站稳，锚定意义，体现我们呈现的沉思或洞察。

在建筑摄影中，人影不应该是一个用于比例的小点。他们应该占据有意义的空间——也许是画面高度的三分之一，有意地定位，他们的姿态或视线引导观者的眼睛，给几何形态赋予人的意义。

在电影化镜头中，人可以是剪影、部分遮挡、在中景——但他们的存在应该感觉有意而有力，不是偶然的。

人的形象提醒观者这一切关于什么：理解人。


## 社交媒体需要冲击力

我们对社交媒体诚实点：如果你的图片不能在半秒内停止滚动，再有品味也没用。

这是"精致"常常失败的地方。设计师创造出微妙而优雅的东西，然后奇怪为什么没有互动。真相很残酷：你在和数百万张图片竞争。你的必须能抓住注意力。

但关键在这里：我们通过专业手段实现冲击力，不是廉价技巧。

**专业冲击力来自**：戏剧性色彩对比（暖对冷）。强烈几何构图。电影化打光。建筑尺度。有目的的大胆饱和度。锚定注意力的人的存在。

**我们避免的廉价技巧**：霓虹渐变。阴影和发光效果。多个竞争色彩。信息图表式布局。合成库存照片。漂浮在空间中的3D渲染对象。

区别？专业冲击力有实质。经得起细看。廉价技巧在缩略图看着好，放大就崩了。

对于封面图像：用呼吸空间构图，用大胆色彩关系，在相关时确保人的存在，确保核心视觉即使小尺寸也清晰可读。但永远不要为了点击牺牲品味——目标是来自正确受众的注意力。


## 什么是糟糕的设计

信息图表式的图标加几何图形堆砌——这是最常见的错误。人们以为这是"数据可视化"，其实只是机械组装，没有思考，没有品味。

过度合成、缺乏质感的元素。浮夸的3D渲染图形，除非主题确实需要。低质量、粗糙、压缩失真的视觉——对内容不尊重。

在线模板设计让所有东西看起来一样。"未来科技"的视觉陈词滥调，创业公司pitch deck的商业模板感——避免所有这些。

还有那些"让它更丰富"而添加的装饰元素——如果一个元素除了填空白之外没有其他作用，它就不应该存在。

但也要避免相反的极端：过于胆小的设计，技术上正确但没有存在感、没有力量、无法吸引人。


## 报告设计

研究报告应该有顶级咨询公司的专业度，配建筑摄影的戏剧性和纪实摄影的人性。

**色彩**：黑、白、灰作为基础。如果需要，可以用单一强调色。没有彩色卡片、背景色块、粗边框。但在展示数据或示例时，摄影可以有完整的电影色彩——克制在于版式和排版，不是压制所有色彩。

**排版**：通过字重渐进建立层级——Regular到Bold。尺寸表示重要性。留白创造呼吸感。对齐创造秩序。最好的排版在你需要读之前是隐形的，需要时就毫不费力。

**内容**：清晰的逻辑流。克制但有力的数据可视化。有尊严地呈现用户访谈引语。清晰严格的来源引用。整体感觉应该是顶级咨询公司遇见人类学田野笔记再遇见建筑摄影画册。


## 持续演进

设计不是一次性的，是持续演进。每季度检查表现数据——点击率、分享率、停留时间。收集用户反馈，评估品牌认知。问：我们是否守住了"越不AI越AI"？

要警惕：电影摄影是否变成了我们的新默认模板？我们是否在安全选择和美学冒险之间保持平衡？那些禁止项是真的在执行还是只是口头说说？

每月做反常规实验。故意打破舒适区，尝试从未用过的视觉形式，小心测试边缘，探索新可能性。然后严格评估：这是否依然符合核心原则？

建立案例库。记录成功和失败。这个为什么成功了？那个为什么不行？可接受的临界点是什么？这些经验应该被记录和分享。

设计指南本身应该演进。新的视觉语言发现，新的禁止项识别，新的判断框架优化。这个文档不是圣经，是工作手册，随着我们的理解深化而深化。


## 质量标准

**视觉冲击力**：它能停止滚动吗？它能抓住注意力吗？它有力量和存在感吗？构图在缩略图和全尺寸都够强吗？

**专业工艺**：摄影或设计工作是否成熟？色彩关系是否有意且有意义？它是否避免了廉价技巧和模板美学？这能放进专业作品集吗？

**品牌一致**：它体现"越不AI越AI"吗？它平衡了权威和人性吗？它可辨识为atypica但不僵化吗？它服务于理解人吗？

**功能成功**：它准确传达内容本质吗？它适合目标受众吗？它跨平台工作吗？细节经得起检查吗？


## 最后

用最有力的人性化方式，呈现最智能的洞察。

这不是风格指南，是视觉传达的哲学。不是限制创意，是把创意引向冲击力。不是固定模板，是在每个具体场景做正确决策的判断框架。

设计应该诚实而有力。诚实面对产品本质，诚实面对我们服务的对象，诚实面对美学判断。有力到能抓住注意力，有深度到值得这份注意力。

在一个AI产品都努力"看起来很AI"用廉价未来感美学的市场里，我们选择"看起来很人"用成熟的专业视觉语言。不是因为这很潮，是因为这符合我们做的事——帮助人们理解人。

我们越少依赖AI视觉陈词滥调，就越能展示AI真正应该做什么：用智能和深度照亮人性理解。
