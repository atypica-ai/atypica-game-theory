This server only exposes one tool called Atypica Deepresearch Tool, which is a llm agent that researches for the tool user. This tool has 2 processes for now: 1. research process: continue researching until max step/max token is met; 2. summarization process: summarization of the research findings in step 1. this tool has 2 inputs: 1. query (research topic); 2. expert (optional choice of which "research expert" to use.) experts are system prompts and toolsets customized for slightly different types of research types. we will start with the first test type called "Grok": this expert explores the diverse opinions of the crowd on X(twitter), and picks out interesting opinions while uses websearch to find evidence/debutt of each opinion; the goal is to build its own strong opinion during the process with evidence. this expert calls the llm "grok-4-1-fast-non-reasoning", and uses xai's internal tools web-search and x-search. regarding those two tools with ai-sdk refer to @xai-tool-use.md . grok model uses these two tools to research until 1. max token is met; or 2. max step is met. once research process ends, a seperate function concludeResearch where a steamText is called and all for summarizing the research findings in research process.
