## v7.x: 企业级协作与数据洞察

### `v7.1.0` — _2025-08-08_

- **✨ 新功能: 团队管理**
  - 新增团队协作空间，支持成员邀请、移除和角色管理。
  - 实现用户身份切换，方便在个人与团队空间中无缝工作。

### `v7.0.0` — _2025-08-01_

- **✨ 核心功能: 构建私有 AI 人设 (AI Persona Import) [Beta]**
  - 上线核心功能 `AI 人设导入`，开启 **Max 套餐用户公测**。
  - 用户可上传自己的访谈记录 (`PDF`, `JSON`, `CSV`)，构建出完全私有的 `真人 AI 人设`。
  - AI 会对上传内容进行多维度分析与打分，生成可交互的 `AI 人设`，并以雷达图等可视化形式呈现分析结果。

## v5.x: 智能报告与体验飞跃

### `v5.1.0` — _2025-05-31_

- **✨ 新功能: 报告内图像生成**
  - 集成 `Google Imagen 4.0` 与 `Midjourney` 模型，支持在研究报告中根据上下文动态生成高质量图像 (2025-05-25)。
- **🚀 性能: Prompt 缓存**
  - 为 `Bedrock Claude` 启用 Prompt 缓存，对历史消息进行哈希计算，将重复请求的 Token 消耗降低 90% (2025-05-29)。
- **🔧 优化: 语音输入**
  - 优化语音识别功能，使用 `Groq Whisper` 模型，实现语音输入的即时转文本，大幅提升交互速度。
  - 修复了因 Vercel AI SDK 导致 `prompt cache` 失效的问题。

### `v5.0.0` — _2025-05-20_

- **✨ 新功能: 研究附件**
  - 用户可在发起研究时上传文档 (`PDF`等) 作为上下文，AI 会在后续的访谈和报告生成中参考这些信息 (2025-05-18)。
- **✨ 新功能: Persona 向量搜索**
  - 数据库集成 `pgvector` 扩展，为 Persona 增加 `Embedding` 向量，实现基于语义的模糊搜索 (2025-05-13)。
- **✨ 新功能: 报告下载 (PDF)**
  - 部署独立的 `html-to-pdf` 服务，支持将研究报告一键下载为高质量的 PDF 文件 (2025-05-12)。
- **🔧 优化: Agent 交互**
  - `Study Agent` 现在可以向 `Interview Agent` 传递更具体的指令和上下文，访谈反馈也更丰富，显著降低了整体 Token 消耗。

## v4.x: 平台架构升级

### `v4.1.0` — _2025-04-30_

- **🔧 平台架构: 数据库迁移**
  - 后端数据库由 `MySQL` 成功迁移至 `PostgreSQL`，为后续的向量搜索和复杂查询打下基础 (2025-04-29)。
  - 通过增加索引解决了迁移初期 `nerd stats` 查询导致的性能问题。
- **🔧 优化: 模型与成本**
  - 引入 `gpt-4o` 模型并支持并行工具调用，有效降低了研究成本 (2025-04-16)。
  - 引入 `Google Gemini 1.5 Flash` 模型，进一步降低特定任务成本 (2025-04-21)。
  - 增加严格的 Tokens 和步骤数限制，避免研究任务意外超时或超支 (2025-04-28)。

### `v4.0.0` — _2025-04-20_

- **💰 商业化: Tokens 计费模式上线**
  - 计费模式从简单的“点数”切换为更精细的 `Tokens` 计费。
  - 正式推出 `Pro` 订阅套餐与新的 `Pricing` 定价页面 (2025-04-20)。
  - 集成 `Stripe`，支持全球用户信用卡支付 (2025-04-09)。
- **🔧 平台架构: 数据库核心重构**
  - 将聊天消息 `ChatMessage` 从主表拆分到独立的表中，解决并发写入时的数据覆盖问题，提升了系统的稳定性和扩展性 (2025-04-12)。
- **✨ 新功能: 多平台数据接入**
  - 接入 `TikTok` 和 `Instagram` 数据源，拓展了消费者洞察的渠道 (2025-04-17)。
