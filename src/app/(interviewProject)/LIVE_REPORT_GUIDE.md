# Live Interview Report Generation Guide

This guide explains how to use the new live report generation feature in Interview Projects.

## Overview

The live report generation feature allows you to watch your interview reports being created in real-time by AI, providing an engaging and transparent experience of the analysis process.

## Features

### 🔴 Live Generation Dialog
- **Real-time Viewing**: Watch the AI write your report as it happens
- **Interactive Interface**: Continue working while generation happens in background
- **Multi-tab Support**: Open the report in a new browser tab during generation
- **Auto-completion**: Dialog automatically closes when generation finishes

### 📊 Smart Report Management
- **Card-based Grid**: Visual report organization with status indicators
- **Hover Actions**: Progressive disclosure of share, view, and delete options
- **Status Tracking**: Clear indicators for generating vs. completed reports
- **Background Polling**: Automatic status updates every 3 seconds

### 🔗 Seamless Sharing
- **Instant Access**: Click generating reports to view live progress
- **Professional Sharing**: Completed reports get beautiful share pages
- **Export Ready**: Print or save as PDF directly from browser

## How to Use

### Step 1: Generate a Report
1. Navigate to your Interview Project details page
2. Ensure you have completed at least one interview session
3. Click the **"Generate Report"** button in the Reports section

### Step 2: Watch Live Generation
1. A dialog window opens showing real-time generation
2. Watch as the AI analyzes your interviews and writes the report
3. Use **"Open in New Tab"** to view in a separate window
4. Close the dialog anytime - generation continues in background

### Step 3: Access Your Report
1. The dialog auto-closes when generation completes
2. Your report appears in the grid with a "Generated" status
3. Hover over completed reports to see action options:
   - **Share**: Open professional sharing page
   - **View**: Open raw report in new tab
   - **Delete**: Remove the report

## Tips & Best Practices

### 🎯 Optimal Usage
- **Quality Interviews**: Longer, more detailed interviews produce richer reports
- **Mixed Perspectives**: Combine human and AI interviews for diverse insights
- **Multiple Reports**: Generate different reports to compare analysis approaches

### ⚡ Performance Tips
- **Background Generation**: Feel free to close the live dialog and continue working
- **Browser Tabs**: Use "Open in New Tab" for better multitasking
- **Network**: Stable internet connection ensures smooth live streaming

### 🛠 Troubleshooting
- **Generation Stuck**: Refresh the page and check the reports grid
- **Empty Reports**: Ensure your interview sessions have substantial conversation
- **Dialog Issues**: Try opening the report in a new tab instead

## Technical Details

### Live Streaming
- Uses Server-Sent Events for real-time updates
- Streams HTML content as it's generated by the AI
- Automatically handles connection recovery

### Background Processing
- Report generation continues even if you close the dialog
- Polling mechanism checks completion status every 3 seconds
- Memory-efficient cleanup prevents resource leaks

### Report Format
- Professional HTML layout optimized for reading and sharing
- Responsive design works on desktop and mobile
- Print-friendly styling for PDF export
- No images - pure text for fast loading

## Keyboard Shortcuts

- **Ctrl/Cmd + Click**: Open report in new tab (on completed reports)
