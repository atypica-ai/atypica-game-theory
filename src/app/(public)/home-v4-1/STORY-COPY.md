# Atypica 2.0 Narrative Script (Home V4.1)

## Hero

Atypica 2.0 is the agent that understands humans.

Most agents are built to do tasks.
Atypica is built to understand decisions.

Execution can be automated.
Judgment must be modeled.

## Two Agent Paradigms

Worker agents optimize output.
They generate assets, write code, and automate workflows.

Understanding agents optimize judgment.
They model motives, values, trade-offs, and context.

If your strategic bottleneck is production speed, worker agents help.
If your strategic bottleneck is knowing why customers choose, delay, switch, or reject, you need understanding agents.

Atypica 2.0 is built for the second problem.

## What Atypica Actually Does

Atypica does not stop at surface answers.
It builds a structured model of subjective decision logic.

The system captures value systems, risk preferences, emotional triggers, decision pathways, and social influence effects.

This moves teams from descriptive analytics (what happened)
to cognitive analytics (why this happened and what may happen next).

## Simulator Mode

Simulator mode reconstructs one concrete decision journey at a person level.
It does not flatten behavior into a population average.

It can replay comparison behavior, hesitation loops, perceived risk, and commitment conditions.

Teams can test counterfactuals:
- What changes if price moves?
- What changes if social proof rises?
- What changes if trust is weak?
- What changes if switching cost feels unclear?

## Researcher Mode

Researcher mode behaves like a trained interviewer with dynamic follow-up.

It supports:
- 1v1 deep interviews
- One-to-many sessions
- Focus groups
- Panel discussions
- Longitudinal behavior tracking

Atypica does not only collect statements.
It maps contradictions, latent concerns, confidence shifts, and preference instability over time.

## Subjective World Model

Atypica’s core output is a decision model, not a transcript dump.

Model dimensions include:
- Value system: what this person optimizes for
- Risk preference: what uncertainty is acceptable
- Emotional trigger: what accelerates or blocks action
- Decision pathway: how options are filtered and compared
- Social influence: whose opinion changes outcomes

This model can be compared across segments,
tracked over time,
and stress-tested under new scenarios.

## Multi-Modal Understanding

No single method is enough.

Atypica integrates:
- AI 1v1 interviews
- AI focus groups
- AI panel simulation
- Persona vector modeling
- Agent behavior simulation

Different methods reveal different truths.
The advantage comes from synthesis, cross-validation,
and conflict resolution across modalities.

Typical users include product, growth, strategy, insights, brand, and innovation teams.

Atypica 2.0 does not decide for humans.
