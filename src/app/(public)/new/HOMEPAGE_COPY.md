## 📝 文案修改指南

### 1. 直接修改流程

1. 在本文档中直接修改相应的文案内容
2. 将修改后的内容更新到 `messages/en-US.json` 文件的对应键值
3. 刷新页面查看效果

### 2. 翻译键值对照表

| 文档位置                 | 翻译键值                           |
| ------------------------ | ---------------------------------- |
| Hero Section subtitle    | `HomePage.HeroSection.subtitle`    |
| Hero Section description | `HomePage.HeroSection.description` |
| Features Section badge   | `HomePage.FeaturesSection.badge`   |
| Demo Section quote       | `HomePage.DemoSection.quote.text`  |
| CTA Section title        | `HomePage.CTASection.title`        |

### 3. 注意事项

- 保持按钮文本简洁（2-4个单词）
- 确保移动端显示效果良好
- 保持技术描述的准确性
- 维持整体语调的一致性
- 测试不同屏幕尺寸下的可读性
